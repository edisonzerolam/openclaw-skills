> 本文件为 agent-planner skill 的坑点库，按需读取。每次规划至少匹配 3 个与方案相关的坑点。
> 维护规则：新坑点追加到末尾，不修改已有条目，版本号递增。

## 使用方法

1. 读取本文档
2. 根据方案特征筛选相关坑点（≥3个）
3. 在方案中引用坑点编号（如"参考T2/T5/T7"）
4. 结合方案风险补充坑点库中未覆盖的新风险
