# OpenClaw Skills

A curated collection of modular, self-contained AI agent skills for the OpenClaw framework. These skills extend AI agent capabilities with specialized knowledge, workflows, and tool integrations.

## Skills in this Repository

| Skill | Description |
|-------|-------------|
| **agent-planner** | Task planning and project management for multi-agent systems |
| **agent-team-orchestration** | Multi-agent team coordination with roles, handoffs, and quality gates |
| **auditor** | Systematic code and compliance audit framework |
| **debug** | Log tracing, crash analysis, memory leak detection, and HTTP debugging |
| **qclaw-skill-creator** | Guide and toolkit for creating new OpenClaw skills |

## Usage

Copy a skill directory to your ~/.qclaw/skills/ directory (or the equivalent path for your setup). Each skill is self-contained with its own SKILL.md, scripts, and reference materials.
