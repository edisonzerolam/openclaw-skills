# 策略总监知识库

> 内容分发策略总监（strategy-director）的本地知识手册。用于全平台分发策略制定和ROI评估。

---

## 1. 全平台分发策略框架

### 平台分类模型

| 平台类型 | 特点 | 代表平台 | 内容策略 |
|----------|------|----------|----------|
| **流量型** | 算法主导，爆发力强 | 抖音/TikTok | 短内容优先，标题党 |
| **社区型** | 社区粘性，长尾强 | 小红书/B站 | 深度内容，垂直圈层 |
| **社交型** | 社交关系，朋友圈传播 | 微信/微博 | 情感共鸣，引发分享 |
| **搜索型** | 搜索流量，长效价值 | YouTube/微信搜一搜 | SEO优化，常青内容 |

### 内容分发优先级决策树

```
内容类型判断
├── 热点/娱乐类 → 优先抖音/TikTok（流量爆发）
├── 知识干货类 → 优先小红书/YouTube（长尾价值）
├── 深度分析类 → 优先B站/公众号（深度用户）
└── 社交传播类 → 优先微信/微博（社交裂变）
```

## 2. ROI评估方法

### 核心ROI指标

$$ROI = \frac{收益}{成本} = \frac{内容带来的可衡量价值}{内容生产成本}$$

**可衡量的收益类型**：
- 广告收入（平台分成、品牌合作）
- 直接转化（引流私域、商品销售）
- 品牌曝光（品牌知名度提升）

### 内容生产成本计算

| 成本项 | 计算方法 |
|--------|----------|
| **人力成本** | 参与人员时薪×工时 |
| **制作工具** | 软件订阅、AI工具成本 |
| **推广费用** | 付费投放金额 |
| **外包费用** | 达人合作/外包制作 |

### ROI评估标准

| ROI倍数 | 评价 | 决策 |
|--------|------|------|
| >5x | 极优质内容 | 加大投入，批量复制 |
| 2~5x | 良好内容 | 持续优化，可适度复制 |
| 1~2x | 一般内容 | 优化方向，降低成本 |
| <1x | 低效内容 | 重新评估内容策略 |

| 内容类型 | 占比 | 目的 |
|----------|------|------|
| **引流内容**（热点、娱乐） | 30% | 流量爆发 |

## 4. 数据复盘方法论

### 核心复盘维度

| 维度 | 指标 | 分析目的 |
|------|------|----------|
| **流量** | 播放/阅读量 | 内容触达能力 |
| **互动** | 点赞/评论/收藏/分享 | 内容质量判断 |
| **转化** | 私域新增/商品点击 | 变现能力 |
| **粉丝** | 净增/取关/活跃度 | 账号健康度 |

### 复盘周期

| 周期 | 内容 | 负责人 |
|------|------|--------|
| **日度** | 关键数据（播放/互动）快速扫视 | 运营专员 |
| **周度** | 详细数据对比，数据驱动优化 | 运营+策略 |
| **月度** | 整体复盘，策略调整 | 策略总监 |
| **季度** | 战略复盘，资源重新配置 | 团队全员 |
