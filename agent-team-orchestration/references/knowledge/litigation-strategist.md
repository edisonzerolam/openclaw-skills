# 诉讼策略师 — 诉讼策略制定知识库

## 一、诉讼策略制定框架

### 1.1 策略制定三要素

```
            诉讼请求
               │
    ┌──────────┼──────────┐
    ↓          ↓          ↓
 法律依据    事实证据    风险评估
    │          │          │
 找法条     固定事实    算风险
    │          │          │
 配请求     配证据      定方案
```

### 1.2 诉讼请求确定方法

```
诉讼请求制定步骤：

Step 1：确定权利基础
   - 合同请求权（合同履行/解除/损害赔偿）
   - 侵权请求权（人身/财产/精神损害）
   - 不当得利请求权
   - 无因管理请求权
   - 物权请求权（返还原物/排除妨害/消除危险）

Step 2：量化请求金额
   - 本金计算
   - 利息/违约金计算
   - 损害赔偿计算
   - 诉讼费用测算

Step 3：排列请求优先级
   - 主请求（核心权利）
   - 备选请求（替代方案）
   - 从请求（利息/费用等）

Step 4：检查请求可行性
   - 诉讼时效是否经过
   - 受诉法院是否有管辖权
   - 是否属于法院主管范围
```

## 二、诉讼时效计算

### 2.1 普通诉讼时效（民法典）

```
法条依据：《民法典》第188条

时效期间：3年
起算点：权利人知道或者应当知道权利受到损害以及义务人之日

特殊起算规则：
- 人身伤害：知道或者应当知道权利受侵害+知道侵害人
- 分期付款：自最后一期履行期限届满之日起算
- 无期徒刑追诉：人民法院、人民检察院或者公安机关应当立案而在通缉后二十年未被追诉
```

### 2.2 特殊诉讼时效

| 类型 | 时效期间 | 法条依据 | 起算点 |
|------|----------|----------|--------|
| 人身伤害赔偿 | 3年 | 《民法典》188条 | 知道+知道责任人 |
| 产品责任 | 3年 | 《产品质量法》45条 | 知道缺陷+知道生产者 |
| 环境损害 | 3年 | 《民法典》188条 | 知道污染+知道侵权人 |
| 航运船舶碰擅 | 2年 | 《海商法》257条 | 事故发生之日 |
| 劳动报酬 | 劳动关系终止后1年 | 《劳动争议调解仲裁法》27条 | 劳动关系终止 |
| 工伤保险待遇 | 1年（仲裁） | 《工伤保险条例》55条 | 事故伤害发生之日/职业病确诊之日 |
| 财产抵押/质押 | 2年主债权诉讼时效结束后 | 《民法典》419条 | 主债权诉讼时效期间届满 |

### 2.3 时效中断与延长

```
中断事由（《民法典》第195条）：
□ 权利人向义务人提出履行请求
□ 义务人同意履行义务
□ 权利人提起诉讼或者申请仲裁
□ 与提起诉讼或者申请仲裁具有同等效力的其他情形
  - 申请支付令
  - 申请诉前财产保全/诉前证据保全
  - 申请追加当事人/第三人参与诉讼

中断效果：期间完成前再次开始起算，重新计算

时效延长（特殊情况）：
- 权利人被控制（人身自由被剥夺）→ 不计算在内
- 未成年人遭受性侵害→ 年满18周岁起算
```

### 2.4 时效抗辩实务要点

```
法条依据：《民法典》第192条

规则：诉讼时效期间届满后，义务人可以提出不履行义务的抗辩
例外：义务人同意履行/义务人已自愿履行（不得请求返还）

举证要点：
- 义务人证明诉讼时效期间已届满
- 债权人证明存在中断/中止事由

注意：
- 法院不得主动适用诉讼时效（当事人未提出抗辩则不审查）
- 一审未提出时效抗辩，二审原则上不得提出（除非新证据）
- 分期债务每一期分别计算时效
```

## 三、举证责任分配

### 3.1 举证责任分配原则

```
基本原则："谁主张，谁举证"（《民事诉讼法》第67条）

例外规则：
- 举证责任倒置（法律规定）
- 举证责任转移（持有证据方不提供）
- 举证责任减轻（证明困难时的酌定减轻）
```

### 3.2 常见纠纷举证责任分配

| 纠纷类型 | 原告（权利主张方）举证 | 被告（抗辩方）举证 | 特殊规则 |
|----------|------------------------|---------------------|----------|
| **合同违约** | 合同成立/生效、履行事实、损失 | 免责事由、减损义务 | — |
| **产品责任** | 产品缺陷、损害事实 | 不存在缺陷/免责事由 | 举证责任倒置（生产者） |
| **医疗损害** | 医患关系、诊疗行为、损害后果 | 诊疗行为无过错/因果关系 | 举证责任倒置（医疗机构） |
| **环境污染** | 污染行为、损害后果 | 不存在因果关系 | 举证责任倒置（污染者） |
| **劳动争议** | 劳动关系、工资/工时、损害 | 合法解除/已支付加班费 | 用人单位举证（工资台账） |
| **侵权纠纷** | 侵权行为、损害后果、因果关系 | 免责事由（正当防卫/紧急避险等） | — |
| **格式条款** | 已交付条款、对方未提示说明 | 已履行提示说明义务 | — |

### 3.3 电子证据举证要点

```
证据类型：
- 微信/QQ聊天记录
- 电子邮件
- 电子合同（电子签名）
- 网页截图
- APP记录

举证要求：
1. 提供原始载体（手机/电脑）
2. 载明证据来源及提取过程
3. 电子签名验证（如有）
4. 必要时申请公证保全

时间戳认证：
- 可通过可信时间戳服务（联合信任TSA）固化证据
- 中国电子证据联盟认证
```

## 四、风险评估矩阵

### 4.1 案件风险评估维度

```
评估维度：
1. 法律风险（胜诉概率）
2. 时间成本（审理周期）
3. 经济成本（诉讼费用+律师费+其他费用）
4. 执行风险（对方履行能力）
5. 声誉风险（公开审理/舆论影响）
```

### 4.2 胜诉概率评估

| 等级 | 概率区间 | 判定标准 |
|------|----------|----------|
| 高 | 70-90% | 事实清楚/法律明确/证据充分 |
| 中 | 40-70% | 存在争议/法律模糊/部分事实待查 |
| 低 | 10-40% | 事实复杂/法律无明确规定/证据不足 |

### 4.3 风险评估问答清单

```
回答以下问题以完成风险评估：

法律风险：
□ 诉讼时效是否经过？
□ 管辖法院是否正确？
□ 请求权基础是否充分？
□ 证据链是否完整？
□ 对方是否有有力抗辩？

经济风险：
□ 诉讼标的金额与诉讼成本比例？
□ 对方是否有履行能力？
□ 对方是否正在转移资产？

程序风险：
□ 是否需要财产保全？
□ 是否需要先履行前置程序（仲裁等）？
□ 上诉/再审可能性？

执行风险：
□ 对方名下财产情况？
□ 是否需要申请强制执行？
□ 是否需要纳入失信名单/限制高消费？
```

## 五、策略方案备选

### 5.1 和解策略

```
适用场景：
- 对方有和解意愿
- 诉讼成本过高
- 维持商业关系
- 执行风险较大

和解要素：
- 明确底线金额/条件
- 制定让步方案
- 设计付款安排
- 约定违约责任
```

### 5.2 调解策略

```
法院调解：
- 立案后向法院申请调解
- 调解书具有强制执行效力
- 诉讼费减半收取

仲裁调解：
- 适用于已达成仲裁协议的纠纷
- 一裁终局，效率高
- 调解结案可规避裁决说理

人民调解：
- 适用于邻里/家庭纠纷
- 调解协议可申请司法确认
- 不收费
```

### 5.3 财产保全策略

```
适用条件：
- 情况紧急
- 对方有转移/隐匿财产可能
- 为执行提供保障

保全类型：
- 诉前保全（紧急时）
- 诉中保全（立案后）
- 执行前保全（判决后执行前）

申请材料：
- 保全申请书
- 担保材料（保函/保证金）
- 财产线索

担保要求：
- 诉前保全：提供等额担保
- 诉中保全：原则上提供担保（法院可酌定）
```
