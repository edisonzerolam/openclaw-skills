# 综合撰写师知识库

> 综合撰写师（synthesis-writer）的本地知识手册。用于A股研报撰写时的格式/评级/方法论引用。

---

## 1. 研报标准格式框架

### 四层结构（A股标准）

```
第一层：宏观环境
第二层：市场策略
第三层：行业/主题
第四层：个股分析
```

### 宏观环境章节

**必须包含**：
1. 宏观经济数据（GDP/CPI/PPI/社融/出口）
2. 货币政策信号（流动性松紧、利率政策）
3. 财政政策导向（赤字率、专项债、政策重点）
4. 海外环境（美联储政策、全球需求）

### 市场策略章节

**必须包含**：
1. 估值水平（PE/PB历史分位、股债收益率比）
2. 资金面（北向资金、融资余额、新发基金）
3. 市场情绪（换手率、波动率、情绪指数）
4. 风格判断（大盘/小盘、价值/成长）

### 行业/主题章节

**必须包含**：
1. 行业景气度（量/价/利三维）
2. 政策环境（支持/中性/限制）
3. 竞争格局（集中度、龙头优势）
4. 估值水平（行业PE/相对沪深300溢价）

### 个股分析章节

**必须包含**：
1. 核心竞争力（护城河来源）
2. 财务表现（成长性、盈利能力、现金流）
3. 估值分析（绝对估值+相对估值）
4. 风险提示（行业风险、公司特有风险）

## 2. 投资评级定义

### A股主流评级体系

| 评级 | 定义 | 操作建议 |
|------|------|----------|
| **强烈推荐** | 未来6个月涨幅>30%，确定性高 | 大幅超配 |
| **推荐** | 未来6个月涨幅15~30% | 适度超配 |
| **谨慎推荐** | 未来6个月涨幅5~15% | 标配 |
| **中性** | 未来6个月涨幅-10~5% | 低配 |
| **谨慎减持** | 未来6个月涨幅-10~-30% | 减配 |
| **减持** | 未来6个月跌幅>30% | 回避 |

### 评级上轨/下轨

$$
上限涨幅 = \frac{目标价 - 当前价}{当前价} \times 100\%
$$

$$
下限跌幅 = \frac{止损价 - 当前价}{当前价} \times 100\%
$$

- 止损价通常设为成本价的 -15% 或估值下限对应的价格

### 评级调整信号

| 调整方向 | 正面信号 | 负面信号 |
|----------|----------|----------|
| 上调 | 业绩超预期、核心驱动浮现、政策加码 | — |
| 维持 | 核心逻辑未变 | — |
| 下调 | 景气下行、竞争恶化、政策收紧 | 业绩低预期、风险事件 |

## 3. 目标价确定方法

### 方法一：DCF估值法

1. 估算3-5年净利润CAGR
2. 计算自由现金流
3. 设定WACC和永续增速
4. 得出DCF估值 → 目标价

### 方法二：相对估值法（PE/PB法）

$$
目标价 = 同类公司PE/PB均值 \times EPS/BPS
$$

**同类公司选择标准**：
1. 主营业务相似度 >70%
2. 所处行业发展阶段相近
3. 市值规模差异 <5倍

### 方法三：向上空间法

$$
向上空间（%）= \frac{目标价 - 现有价格}{现有价格} \times 100\%
$$

$$
向下空间（%）= \frac{现有价格 - 止损价}{现有价格} \times 100\%
$$

**优质标的特征**：向上空间>50%，向下空间<15%（盈亏比>3:1）

## 4. 催化剂分析法

### 催化剂分类

| 催化剂类型 | 时间窗口 | 预期影响幅度 |
|------------|----------|--------------|
| 业绩公告（季报/年报） | 公告前后1周 | 超预期±10~20% |
| 重大政策出台 | 即时反应 | 行业涨幅15~30% |
| 重大合同/订单 | 公告当日 | 个股涨幅5~15% |
| 行业景气拐点 | 1-3个月内 | 行业涨幅20~50% |
| 管理层变动 | 即时反应 | 影响不确定 |
| 并购/重组 | 停牌前后 | 视方案而定 |

### 催化剂评估框架

**评估维度**：
1. **确定性**：事件发生概率（高/中/低）
2. **时间确定性**：是否有时限（明确日期/模糊）
3. **幅度影响**：对业绩/估值的量化影响
4. **市场预期差**：当前股价是否已反映

**组合筛选**：
- 高确定性 + 中高幅度 + 未充分反映 → 优先配置
- 低确定性 + 模糊时间 → 谨慎对待

## 5. 研报写作规范

### 正文结构

```
1. 核心观点（200字以内）：最核心的1-2个判断
2. 正文各章节（按四层结构展开）
3. 风险提示（至少3条）
4. 附录：数据来源、估值模型参数
```

### 数据来源标注规范

| 数据类型 | 标注格式 |
|----------|----------|
| 公司公告 | 来源：公司公告，日期 |
| 宏观数据 | 来源：国家统计局，发布日期 |
| 行业数据 | 来源：中汽协/工信部/行业协会，月份 |
| 行情数据 | 来源：Wind/东方财富，日期 |

### 利益披露模板

```
声明：本报告仅代表作者个人观点，不代表所在机构。报告中的公司为作者关注范围内，不构成投资建议。报告撰写者未持有文中所提股票。
```
