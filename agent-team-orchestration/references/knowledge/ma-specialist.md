# 并购法务专家知识库

> 企业法务并购法务专家（ma-specialist）的本地知识手册。用于并购交易结构设计、法务尽职调查和交易文件审查。

---

## 1. 并购交易结构

### 三种基本交易结构

| 结构 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **资产收购** | 风险隔离，只收购特定资产 | 税务负担较重（需缴纳增值税、土地增值税等） | 收购特定资产/业务，而非整个公司 |
| **股权收购** | 税负较低，原公司壳保留 | 承继全部负债（隐藏负债风险） | 收购目标公司股权 |
| **合并** | 完全整合，规模效应 | 决策复杂，审批要求高 | 强强联合或消灭竞争对手 |

### 特殊交易结构

**VIE架构**（可变利益实体）：
- 用于外资限制/禁止行业（互联网、教育等）
- 通过协议控制而非股权控制实现合并报表
- 存在政策风险（涉及监管态度）

**SPAC并购**：
- 先上市SPAC，再并购目标公司
- 适合估值明确、确定性高的公司

## 2. 法务尽职调查清单

### 必备调查事项

| 调查模块 | 核心问题 | 关注文件 |
|----------|----------|----------|
| **公司设立** | 设立程序是否合规 | 营业执照、公司章程、股东名册 |
| **股东权益** | 股权清晰无争议 | 股权协议、增资协议、股东会决议 |
| **重大合同** | 是否存在限制性条款 | 贷款协议、担保合同、重大采购/销售合同 |
| **债权债务** | 债务真实性和完整性 | 银行贷款、民间借贷、应付账款 |
| **劳动人事** | 用工合规性 | 劳动合同、社保缴纳记录、劳动仲裁记录 |
| **知识产权** | 权属清晰无侵权 | 专利/商标/著作权证书、许可协议 |
| **诉讼仲裁** | 潜在负债 | 诉讼清单、仲裁记录、处罚记录 |
| **合规资质** | 经营许可是否齐全 | 行业特定资质、行政审批文件 |

### 重大风险信号

| 风险信号 | 风险等级 | 建议 |
|----------|----------|------|
| 股权代持未清理 | 高 | 要求在交割前清理完成 |
| 重大未披露诉讼 | 高 | 要求设立 escrow/赔偿条款 |
| 土地使用权瑕疵 | 中高 | 要求提供补偿或调整交易对价 |
| 核心知识产权存在第三方共有 | 中 | 要求补充协议明确各方权利 |

## 3. 交易文件核心条款

### 股权转让协议必备条款

| 条款 | 内容 | 谈判重点 |
|------|------|----------|
| **陈述与保证** | 卖方对目标公司状态的声明和保证 | 卖方保证的广度和深度 |
| **交割条件** | 先决条件满足后才能交割 | 监管审批、客户同意等 |
| **交易对价** | 股权转让价格/支付方式 | 资金托管、分期支付 |
| **赔偿条款** | 违反陈述保证的赔偿责任 | 赔偿上限/下限/期限 |
| **竞业限制** | 卖方/管理层竞业禁止 | 范围/期限/补偿 |
| **保密条款** | 信息保密义务 | 范围/期限/违约责任 |

### 赔偿条款关键参数

| 参数 | 常见约定 | 说明 |
|------|----------|------|
| **赔偿上限** | 交易对价的10~30% | 过高可能影响卖方意愿 |
| **免赔额** | 交易对价的0.5~1% | 小额赔偿不触发 |
| **赔偿期限** | 交割后1~3年 | 税务/诉讼风险可能更长 |
| **赔偿触发条件** | 重大不利影响（MAE） | 需明确定义MAE标准 |

## 4. 监管审批要求

### 反垄断审查（经营者集中）

**申报标准**（同时满足）：
1. 参与集中的经营者全球营业额>100亿元
2. 至少两个经营者中国境内营业额>20亿元
3. 所有经营者中国境内营业额>4亿元

**审查流程**：
```
申报 → 立案 → 初步审查（30日）→ 进一步审查（90日）→ 批准/禁止
```

**附条件批准**：经营者可在满足一定条件（如剥离特定业务）后获得批准。

### 外商投资审查

**外资准入负面清单**（2022年版）：
- 禁止投资：武器弹药、新闻机构等
- 限制投资：增值电信业务（需合资且中方控股）、医疗机构等
- 鼓励投资：制造业升级、节能环保等

### 上市公司重大资产重组

| 要项 | 说明 |
|------|------|
| **重组标准** | 购买/出售资产达到一定比例（资产净额/营业收入/净利润） |
| **停牌制度** | 筹划重大事项可申请停牌，不超过10个工作日 |
| **信息披露** | 重组预案、重组报告书分阶段披露 |
| **审批程序** | 交易所问询→股东大会→证监会审批（如需） |
