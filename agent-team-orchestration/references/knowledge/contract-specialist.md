# 合同专家知识库

> 企业法务合同专家（contract-specialist）的本地知识手册。用于商业合同审查和谈判。

---

## 1. 商业合同审查要点

### 合同类型与审查重点

| 合同类型 | 核心条款 | 高风险点 |
|----------|----------|----------|
| **买卖合同** | 标的交付/验收/付款 | 质量标准/验收期限/退货条件 |
| **租赁合同** | 租金/租期/押金 | 提前退租条款/优先购买权 |
| **委托合同** | 委托范围/报酬/报告义务 | 报告标准/解除委托限制 |
| **合作合同** | 合作方式/收益分配/保密 | 收益分配算法/竞业限制/知识产权归属 |
| **投资合同** | 投资条件/估值调整/退出 | 清算优先权/拖带权/反稀释条款 |

### 合同谈判底线

| 条款 | 可接受底线 | 红线条款 |
|------|-----------|----------|
| 付款条件 | 验收后30天内 | 预付款超过50%且无履约保障 |
| 违约责任 | 损失赔偿上限为合同总额150% | 无上限或超过合同总额300% |
| 争议解决 | 被告所在地法院 | 强制仲裁或境外仲裁 |
| 知识产权 | 共有或被授权 | 独占许可给对方且无返还条款 |
| 保密条款 | 2-3年保密期 | 永久保密或过宽范围 |

## 2. 合同管理流程

### 合同全生命周期

```
合同谈判 → 合同起草 → 内部审批 → 用印签署 → 履行监控 → 到期处理
```

### 各阶段注意事项

**谈判阶段**：保留谈判记录和往来邮件，作为合同解释依据。

**起草阶段**：
- 使用公司标准合同模板（降低风险）
- 对方修改意见用红色标注，逐条确认

**审批阶段**：
- 根据合同金额设置审批权限（避免超权限签署）
- 法律/财务会签：超过50万合同需要法务+财务双签

**履行监控**：建立台账，关键节点提前预警（付款日/到期日/续约期）。

## 3. 履约监控方法

### 履约监控清单

| 监控项 | 我方义务 | 对方义务 |
|--------|----------|----------|
| **付款** | 按合同约定付款，保留凭证 | 监控对方是否按时付款 |
| **交付** | 按时交付，留存交付凭证 | 监控对方是否按质交付 |
| **验收** | 按期验收，提出质量异议 | 对方超期不答复视为验收通过 |
| **保密** | 保密信息不泄露 | 监控对方保密义务履行 |

### 预警机制

| 预警级别 | 触发条件 | 处理方式 |
|----------|----------|----------|
| 黄色预警 | 付款/交付到期前7天 | 提醒对方准备 |
| 橙色预警 | 付款/交付已到期 | 发送书面催告函 |
| 红色预警 | 催告后15天仍未履行 | 启动法律程序评估 |
