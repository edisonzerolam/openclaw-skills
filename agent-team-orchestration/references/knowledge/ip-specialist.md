# 知识产权专家知识库

> 企业法务知识产权专家（ip-specialist）的本地知识手册。用于专利/商标/著作权侵权判定和知识产权尽职调查。

---

## 1. 专利/商标/著作权基础概念

### 专利类型

| 类型 | 保护对象 | 保护期 | 申请门槛 |
|------|----------|--------|----------|
| **发明专利** | 新技术方案（方法、产品） | 20年 | 创造性、新颖性、实用性 |
| **实用新型专利** | 产品形状/结构新设计 | 10年 | 新颖性+实用性，无需实质性审查 |
| **外观设计专利** | 产品外观设计 | 15年（2021年后改为15年） | 新颖性+区别性 |

### 商标权

- **注册制**：中国采用注册在先原则（先注册先得）
- **保护期**：10年，可无限续展（每次10年）
- **驰命商标**：相关公众所熟知的商标，可获得跨类保护

### 著作权（版权）

- **自动取得**：作品完成即享有，无需登记
- **保护期**：个人作品：作者终生+死后50年；法人作品：发表后50年
- **登记意义**：确权证据，便于维权

## 2. 侵权判定方法

### 专利侵权判定

**判断步骤**：
1. 确定专利保护范围（权利要求书解读）
2. 确定被诉产品/方法的技术特征
3. 技术特征逐一对比

**等同侵权原则**：
- 被诉产品以基本相同的手段，实现基本相同的功能，达到基本相同的效果
- 即使不完全一样，也可能构成侵权

### 商标侵权判定

**相同/近似商标判断**：
- 音、形、义任一方面近似
- 可能导致公众混淆

**相同/类似商品判断**：
- 商品功能、用途、生产部门、销售渠道相同
- 消费群体相同

### 著作权侵权判定

**两步法**：
1. **实质性相似**：作品的整体或核心部分相似
2. **接触可能性**：被诉方有机会接触到原作品

## 3. 知识产权尽职调查

### 调查清单

| 调查项 | 方法 | 重点关注 |
|--------|------|----------|
| **专利权属** | 专利局查询权属人 | 共有专利、职务发明、许可限制 |
| **商标权属** | 商标局查询 | 是否有许可、是否即将到期 |
| **著作权权属** | 版权登记查询、作品底稿 | 是否有共有、许可 |
| **权属完整性** | 许可协议审查 | 是否完整许可（独占/排他/普通） |
| **侵权风险** | 自由实施分析（FTO） | 是否侵犯他人知识产权 |
| **价值评估** | 市场价值、法律状态 | 专利有效性、商标续展情况 |

### FTO分析（自由实施分析）

目的：评估产品是否可自由实施，不侵犯他人专利权。

**分析步骤**：
1. 识别产品技术方案中的关键专利
2. 筛选同族专利，判断保护范围
3. 对比产品技术特征与专利权利要求
4. 评估侵权风险及规避可能性

## 4. 许可协议核心条款

### 许可类型

| 类型 | 许可方权利 | 被许可方 |
|------|-----------|----------|
| **独占许可** | 不得自己使用，不得许可第三方 | 唯一被许可人 |
| **排他许可** | 不得许可第三方，可自己使用 | 唯一被许可人 |
| **普通许可** | 可自己使用，可许可多个第三方 | 多个被许可人之一 |

### 许可协议必备条款

1. **许可范围**：地域/期限/使用方式
2. **许可类型**：独占/排他/普通
3. **费用条款**：入门费+里程碑+royalties
4. **改进条款**：改进技术的归属和回授
5. **担保条款**：许可方对权利有效性的担保
6. **侵权条款**：被许可方发现侵权时的通知义务
