# 宏观分析师知识库

## 一、货币政策工具

### 1.1 中期借贷便利（MLF）

**定义**：央行向符合宏观审慎要求的金融机构提供中期基础货币的货币政策工具。

**操作频率**：每月月中操作一次，通常在每月15日左右。

**公式**：
- **投放规模** = 到期量 + 新增量
- **利率传导**：MLF利率 → LPR利率 → 贷款利率

**关键指标**：
| 指标 | 阈值/判断标准 |
|------|--------------|
| MLF利率变动 > 5bp | 货币政策收紧信号 |
| MLF投放规模环比↑30% | 货币政策宽松信号 |
| MLF余额同比↑20% | 银行间流动性充裕 |

### 1.2 常备借贷便利（SLF）

**定义**：央行针对地方法人金融机构提供的隔夜/7天/1个月短期流动性支持。

**利率走廊上限**：SLF利率 = 利率走廊上限（7天SLF约3.05%）

**公式**：
$$利率走廊宽度 = SLF利率 - 活期存款利率$$

### 1.3 逆回购

**定义**：央行向一级交易商买入有价证券、投放流动性的操作。

**常用期限**：7天、14天、28天、91天

**核心指标**：
- **逆回购净投放量** = 投放量 - 到期量
- **7天逆回购利率** = 政策利率风向标

## 二、财政政策指标

### 2.1 消费者物价指数（CPI）

**公式**：
$$CPI = \frac{\text{报告期一篮子商品价格}}{\text{基期一篮子商品价格}} \times 100$$

**A股映射关系**：
| CPI区间 | 市场含义 | 利好板块 |
|---------|---------|---------|
| < 0% | 通缩风险 | 消费、黄金 |
| 0%-2% | 温和通胀 | 消费、地产 |
| 2%-3% | 适中通胀 | 银行、周期 |
| > 3% | 通胀压力 | 原材料、资源 |

### 2.2 生产者物价指数（PPI）

**公式**：
$$PPI = \frac{\text{工业品出厂价格}}{\text{基期工业品价格}} \times 100$$

**公式**：
$$PPI同比 = \frac{\text{本月PPI} - \text{去年同期PPI}}{\text{去年同期PPI}} \times 100\%$$

**A股映射**：
- PPI上行 + 需求旺盛 → 顺周期板块（钢铁/化工/有色）利好
- PPI下行 → 中游制造利润承压

### 2.3 国内生产总值（GDP）

**支出法公式**：
$$GDP = C + I + G + (X - M)$$

其中：C=消费、I=投资、G=政府购买、X=出口、M=进口

**A股增速关系**：
$$A股净利润增速 \approx f(GDP增速) = 1.5 \times GDP增速 + 误差项$$

## 三、利率传导机制

### 3.1 利率传导路径

```
央行政策利率
    ↓
银行间市场利率（DR007/Shibor）
    ↓
银行负债成本
    ↓
贷款市场报价利率（LPR）
    ↓
实体经济融资成本
```

### 3.2 关键利率指标

**DR007（银行间7天质押式回购利率）**：
| 区间 | 市场含义 |
|------|---------|
| < 7天逆回购利率 - 20bp | 流动性过度充裕 |
| 7天逆回购利率 ± 20bp | 流动性合理适度 |
| > 7天逆回购利率 + 20bp | 流动性紧张 |

**Shibor 3M**：
- 与AAA+级中期票据利率高度相关
- 是银行间资金面的核心锚定利率

## 四、汇率对A股影响模型

### 4.1 汇率传导公式

$$A股指数收益率 = \alpha + \beta_1 \times 人民币汇率变化率 + \beta_2 \times 风险偏好 + \varepsilon$$

### 4.2 关键阈值

| 人民币汇率 | 市场影响 |
|-----------|---------|
| USD/CNY < 7.0 | 出口承压，外资流入 |
| USD/CNY 7.0-7.3 | 正常区间 |
| USD/CNY > 7.3 | 贬值预期，外资流出压力 |

### 4.3 北向资金汇率敏感度

$$北向资金净流入 = f(人民币升值预期, 美元指数, A股相对估值)$$

## 五、美联储加息对A股传导路径

### 5.1 传导路径图

```
美联储加息 → 美元指数走强 → 人民币贬值压力
    ↓
中美利差收窄 → 资本外流 → 北向资金流出
    ↓
A股流动性收紧 + 风险偏好下降
    ↓
估值承压（成长股 > 价值股）
```

### 5.2 关键监测指标

| 指标 | 预警阈值 |
|------|---------|
| 中美10Y国债利差 | < 50bp 预警 |
| 美元指数（DXY） | > 105 警惕 |
| 北向资金单日流出 | > 100亿 预警 |
| 离岸人民币汇率 | > 7.5 预警 |

### 5.3 美联储加息对不同板块影响

- **利空板块**：科技股、高估值成长股、北向资金重仓股
- **利好板块**：出口链（美元计价收入）、资源股、抗通胀黄金
