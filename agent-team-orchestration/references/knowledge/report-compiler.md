# 报告撰写师知识库

> 报告撰写师（report-compiler）的本地知识手册。用于法律咨询报告撰写时的格式/规范引用。

---

## 1. 法律咨询报告标准格式

### 报告基本框架

```
一、案情摘要
二、法律分析
三、法律建议
四、风险提示
五、附录（证据材料清单、法律依据）
```

### 案情摘要写法

**要素清单**（STAR结构）：
- **Situation（背景）**：当事人、时间、地点、事件起因
- **Task（任务）**：争议焦点或法律需求
- **Action（行为）**：各方已采取的行动
- **Result（结果）**：当前状态或损失

**写法要求**：
- 客观中立，不添加主观判断
- 争议焦点应归纳为法律问题（如"劳动合同是否有效解除"）
- 字数控制：一般案件500-1000字

### 法律分析写法

**结构**：
1. 法律关系认定（是什么类型的法律关系）
2. 适用法律依据（具体法条）
3. 构成要件分析（逐项比对）
4. 结论（权利/义务/责任是否成立）

**法条引用格式**：
> 《中华人民共和国民法典》第XXXX条第X款："XXXXX"
> 释义：[简要说明法条含义]

## 2. 法律意见书写作规范

### 法律意见书与咨询报告的区别

| 项目 | 法律咨询报告 | 法律意见书 |
|------|------------|----------|
| 正式程度 | 较低 | 高（正式法律文书） |
| 使用场景 | 客户咨询答复 | 重大决策依据 |
| 依据 | 事实+法律 | 事实+法律+尽职调查 |
| 签署 | 可不盖章 | 律师签名+律所盖章 |
| 效力 | 参考性 | 具有一定法律效力 |

### 法律意见书结构

```
标题：关于[事项]的法律意见书
编号：[年份]XX律意见第XX号
出具机构：XXX律师事务所
出具律师：[律师姓名+执业证号]
出具日期

一、委托事项
二、事实依据
三、法律分析
四、法律结论
五、律师声明
律师签名：      日期：
（律所盖章）
```

## 3. 行动建议的具体性标准

### 建议质量分级

| 级别 | 描述 | 示例 |
|------|------|------|
| **L1（模糊）** | 方向性建议，不可执行 | "建议加强合同管理" |
| **L2（较清晰）** | 有方向，需要自行解读 | "建议在签约前核查对方资质" |
| **L3（具体可执行）** | 有明确步骤，对方可直接执行 | "建议发送书面催告函（附模板），对方15日内未回复则启动诉讼" |
| **L4（最优）** | L3 + 时间节点 + 责任人 | "当事人于XX日前向我方发送催告函；若对方15日内未回复，我方于XX日提起诉讼" |

### L3级建议标准

**五要素**：谁（who）、做什么（what）、什么时间（when）、在哪里（where）、做到什么程度（how）。

## 4. 风险提示规范

### 风险分级体系

| 级别 | 含义 | 概率/影响 |
|------|------|----------|
| **高风险** | 可能导致重大损失或法律责任 | 发生概率>30%或损失重大 |
| **中风险** | 可能导致一定损失或不便 | 发生概率10~30% |
| **低风险** | 影响较小，可接受或可控 | 发生概率<10% |

### 风险提示写法

**格式**：
```
[风险N]：[标题]
- 风险描述：[客观描述风险内容]
- 影响程度：[对委托人可能造成的影响]
- 应对建议：[如何降低风险或做好准备]
```

## 5. 证据材料清单规范

### 民事诉讼证据清单（通用）

| 证据类型 | 常见证据 | 举证要点 |
|----------|----------|----------|
| 书证 | 合同、发票、函件、转账记录 | 原件+复印件，提供原件供核对 |
| 物证 | 物品损坏照片、产品实物 | 照片需有原始载体和时间戳 |
| 视听资料 | 录音、录像、微信截图 | 原始载体未经编辑，提供原始载体 |
| 电子数据 | 邮件、微信记录、网页截图 | 建议进行公证固证，保留完整上下文 |
| 证人证言 | 证人出庭作证 | 写明证人身份、与当事人关系、感知事实 |
| 当事人陈述 | 庭审陈述、书面说明 | 陈述应具体，避免模糊表述 |

### 证据清单制作规范

- 每项证据按编号排列
- 注明证据名称、证明内容、证据来源
- 原件/复印件标注清楚
- 需提供"证据目录"方便法院/对方查阅
