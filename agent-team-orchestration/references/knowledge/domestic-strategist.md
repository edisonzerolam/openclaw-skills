# 国内平台策略师知识库

> 国内平台策略师（domestic-strategist）的本地知识手册。用于抖音/小红书/微信/微博运营策略和投放分析。

---

## 1. 各平台运营规范

### 抖音运营核心参数

| 参数 | 标准值 | 说明 |
|------|--------|------|
| **视频时长** | 15秒~3分钟 | 知识类建议1~3分钟 |
| **最佳完播率** | >50% | 低于30%很难进入下一级流量池 |
| **互动率目标** | >5% | 点赞+评论+分享/播放量 |
| **发布频率** | 每天1~2条 | 新账号冷启动期建议每天1条 |
| **标签数量** | 3~5个 | 与内容高度相关 |

### 小红书运营核心参数

| 参数 | 标准值 | 说明 |
|------|--------|------|
| **正文字数** | 300~800字 | 过多阅读疲劳 |
| **图片数量** | 3~9张 | 封面+正文图片 |
| **话题标签** | 5~10个 | 精准长尾词 |
| **互动率目标** | >3% | CES评分制 |
| **发布时间** | 08:00/12:00/18:00 | 早中晚三个黄金点 |

### 微信生态运营

**公众号**：
- 图文打开率行业均值 2~5%
- 标题决定80%的打开率
- 原创标识有流量倾斜

**视频号**：
- 算法权重：社交关系>兴趣推荐
- 朋友圈传播是关键放大器
- 直播+短视频联动效果最佳

## 2. 投放策略（千川/薯条）

### 抖音千川投放

| 投放目标 | 适用场景 | 出价建议 |
|----------|----------|----------|
| **短视频带货（支付）** | 直接转化 | 客单价×20~30% |
| **短视频种草（互动）** | 品牌曝光 | CPE 0.5~2元 |
| **直播带货** | 即时转化 | 直播间GMV×15~25% |
| **粉丝提升** | 账号冷启动 | 0.1~0.5元/粉丝 |

### 千川投放ROI标准

| ROI倍数 | 评价 | 建议 |
|--------|------|------|
| >3x | 优秀 | 加投 |
| 1.5~3x | 良好 | 维持或小幅加投 |
| <1.5x | 亏损 | 优化素材或暂停 |

### 小红书聚光（薯条升级版）

| 投放目的 | 出价方式 | 参考CPE |
|----------|----------|----------|
| **阅读量** | CPE | 0.3~1元 |
| **私信** | CPC | 3~8元/私信 |
| **商品访问** | CPC | 1~3元 |

## 3. 达人合作评估

### 达人筛选标准

| 维度 | 评估指标 | 权重 |
|------|----------|------|
| **数据真实性** | 互动率/粉丝水分检测 | 30% |
| **粉丝画像** | 年龄/性别/地域匹配度 | 25% |
| **内容质量** | 作品质量/调性匹配 | 25% |
| **商业口碑** | 历史合作反馈 | 20% |

### 达人报价参考标准

$$达人报价 = 粉丝量 \times 0.1\sim0.3（元）$$

| 粉丝量级 | 参考报价 | 说明 |
|----------|----------|------|
| KOC（<1万） | 0~500元/条 | 真实种草，转化好 |
| KOL（1~10万） | 500~5000元/条 | 区域达人 |
| 腰部（10~100万） | 5000~5万元/条 | 垂直领域 |
| 头部（100万+） | 5万+/条 | 全网知名度 |

## 4. 直播带货要点

### 直播核心指标

| 指标 | 计算方式 | 行业均值 |
|------|----------|----------|
| **观看人数** | 同时在线峰值 | - |
| **GMV** | 成交总额 | - |
| **客单价** | GMV/订单数 | 50~200元 |
| **转化率** | 订单数/观看人数 | 1~5% |
| **退货率** | 退货订单/总订单 | 10~30% |

### 直播引流策略

**预热阶段**（直播前1~7天）：
- 短视频预告（每天1~2条）
- 粉丝群预热
- 朋友圈/微博预热

**直播中**：
- 前30分钟话术引导关注
- 爆品限时秒杀拉动互动
- 引导分享朋友圈
