# 刑事合规与反腐败指南

> 本指南旨在为企业刑事合规体系建设提供法规依据与实务操作指引，聚焦中国刑事合规不起诉制度、美国 FCPA、OFAC 制裁及中国刑法核心条款对比。

---

## 一、中国刑事合规不起诉制度

### 1.1 制度起源与试点历程

| 时间 | 里程碑事件 |
|------|-----------|
| 2018年 | 最高检启动企业合规改革第一批试点（北京、上海等地） |
| 2020年 | 最高检发布《关于开展企业合规改革试点工作的方案》 |
| 2021年 | 扩大试点至全国10个省市 |
| 2022年 | 《涉案企业合规建设、评估和审查办法（试行）》正式落地 |
| 2023年至今 | 全国铺开，形成"合规整改 + 第三方监督评估"机制 |

### 1.2 核心法规依据

**《刑事诉讼法》第282条（2018年修正）**
> 对于未成年人犯罪案件，符合条件的，可以作出不起诉决定。**（企业合规不起诉援引该条款的类推适用）**

**最高检《关于开展企业合规改革试点工作方案的补充通知》（2021）**
- 适用对象：民营企业及法定代表人、实际控制人
- 适用罪名：商业贿赂、逃税骗税、非法集资、污染环境等
- 前提条件：涉案企业承诺合规整改、认罪认罚

### 1.3 合规不起诉运作流程

```
案件受理 → 评估是否适用合规整改 → 企业提出合规整改方案
        → 检察机关审查备案 → 企业执行整改（3-12个月）
        → 第三方组织评估 → 检察机关听证 → 决定起诉/不起诉
```

### 1.4 合规整改核心要素（参照《企业合规建设评估办法》）

1. **合规组织架构**：设立合规委员会，明确职责
2. **合规管理制度**：反腐败、反商业贿赂、财务管理、出口管制等
3. **风险识别与评估**：定期开展合规风险评估
4. **内部举报机制**：设立匿名举报渠道
5. **第三方审计**：聘请外部律师/会计事务所进行独立审计
6. **持续监督与改进**：合规培训、年度审查

## 二、FCPA（美国反海外腐败法）核心条款

### 2.1 立法背景

FCPA 于 1977 年制定，主要针对美国本土企业及在美上市公司，管制向外国官员行贿的行为。违规后果极其严重，包括刑事罚款（自然人最高25万美元、法人最高200万美元）、监禁及民事处罚。

### 2.2 适用范围

| 主体类型 | 是否适用 | 说明 |
|---------|---------|------|
| 美国发行人（SEC registered） | ✅ | 包括其全球子公司 |
| 美国自然人/法人 | ✅ | 全球管辖 |
| 在美国领土范围内行为 | ✅ | 属地管辖 |
| 使用美国邮件/支付系统 | ✅ | 工具管辖 |
| 外国公司在美上市 | ✅ | 需遵守SEC报告义务 |

**注**：为中国客户在境外的合规培训，若涉及使用美国银行系统或邮件系统，可能触发FCPA管辖。

### 2.3 行贿条款核心要素（Anti-Bribery Provisions）

**禁止内容**（15 U.S.C. § 78dd-1 等）：
- 给付、承诺给付或授权给付任何有价值之物
- 给付对象：外国官员、外国政府机构、国际组织官员
- 目的：获取、维持业务或获得不正当优势

**关键抗辩/例外**：
- **加速支付抗辩（Facilitation Payment）**：仅用于加快外国官员履行其正常职责的小额给付（如政府服务费、通关费），上限通常约100美元，FCPA明确允许但不鼓励
- **当地法律抗辩**：若能证明该给付在外国官员本国法律中合法，需逐案分析

### 2.4 会计条款（Accounting Provisions）

FCPA 的会计条款（15 U.S.C. § 78m）要求：
- 保持准确、完整的账簿、记录和账户
- 建立内部控制系统，防止不当支出
- 对子公司实施充分监督
- **最大弱点**：许多中国子公司通过"小金库"或虚假合同报销行贿支出，账外循环，触犯会计条款

**违规后果**：
- SEC 可对上市公司处以民事罚款
- 司法部可提起刑事诉讼
- 罚金可达行贿收益的2倍或2000万美元（取其高者）

### 2.5 常见违规场景（对中国企业的特殊风险）

| 场景 | 风险说明 |
|------|---------|
| 聘请东道国官员亲属担任顾问 | 通过顾问费变相给付，回避直接行贿 |
| 政府项目"咨询费" | 名义上合规，实则贿赂中间人 |
| 学术赞助/慈善捐赠 | 通过公益基金向官员关联机构输送利益 |
| 加速通关/许可证 | 虽为小额加速费，若金额大或频繁仍构成违规 |
| 在美上市公司收购中国企业 | 继承目标公司的FCPA违规历史，产生连带风险 |

## 三、OFAC 出口管制与制裁

### 3.1 制裁法律基础

- **IEEPA**（国际紧急经济权力法，50 U.S.C. § 1701）：总统宣布国家紧急状态后，对特定国家/实体实施冻结和管制
- **OFAC**（外国资产控制办公室）：负责执行美国制裁，隶属财政部

### 3.2 SDN 列表（Specially Designated Nationals）

**核心要点**：
- SDN 列表包括被制裁个人、实体、国家
- **美国人**（US Person）不得与 SDN 进行任何交易
- 冻结在美资产义务
- **50%规则**：由一个或多个SDN合计持有50%以上权益的实体，自动成为被冻结对象（即使实体本身不在SDN列表上）

**次级制裁（Secondary Sanctions）**：
- 针对非美国人的第三级制裁风险
- 如与伊朗伊斯兰革命卫队（IRGC）关联实体交易，欧洲银行可能面临次级制裁
- 中国企业若在伊朗、朝鲜、俄罗斯受制裁行业开展业务，需高度关注次级制裁风险

### 3.3 99%规则

指 OFAC 在评估是否将某实体纳入 SDN 列表时的持股门槛：
- 若一个实体的股权由一个或多个被禁主体持有 **超过99%，该实体也将被纳入SDN列表**
- 此规则对通过多层持股结构规避制裁的行为尤为关键

### 3.4 常见 OFAC 制裁项目（对中国企业影响最大）

| 制裁项目 | 国家/地区 | 中国企业风险点 |
|---------|----------|--------------|
| 伊朗制裁 | 伊朗 | 能源、航运、银行、敏感商品出口 |
| 朝鲜制裁 | 朝鲜 | 劳务输出、煤炭、铁矿石贸易 |
| 俄罗斯制裁 | 俄罗斯 | 金融、技术、军事终端用户 |
| SDNTK（毒品） | 特定国家 | 化学原料出口 |

### 3.5 合规建议

1. 对交易对手开展 **OFAC 筛查**（建议使用World-Check、Medius等商业数据库）
2. 对涉俄交易建立 **最终用户尽职调查（End-User Due Diligence）**
3. 在合同中加入 **OFAC 合规条款和陈述保证**
4. 建立内部制裁合规 SOP，定期培训

## 四、中国《刑法》第163条 vs 第385条

### 4.1 条文对比

| 项目 | 非国家工作人员受贿罪（第163条） | 受贿罪（第385条） |
|------|------------------------------|-----------------|
| 犯罪主体 | 公司、企业或其他单位工作人员（非体制内） | 国家工作人员 |
| 主体认定标准 | 国有公司、企业委派至非国有公司从事公务的人员以国家工作人员论 |
| 刑罚标准 | 三年以下 → 三年以上十年以下（数额巨大或情节严重） → 十年以上或无期（数额特别巨大） | 三年以下 → 三年以上十年以下 → 十年以上或无期 → 死刑（特别严重情节） |
| 量刑档次 | 受贿数额较大/巨大/特别巨大（参照两高司法解释） | 同上，数额标准相同 |
| 适用范围 | 民营企业、外资企业、混合所有制企业 | 各级政府机关、国有企业、事业单位 |

### 4.2 司法解释关键数字

依据 **两高《关于办理贪污贿赂刑事案件适用法律若干问题的解释》（2016）**：

| 数额档次 | 受贿罪/非国家工作人员受贿罪 |
|---------|--------------------------|
| 数额较大 | 3万元以上 |
| 数额巨大 | 20万元以上 |
| 数额特别巨大 | 300万元以上 |

### 4.3 实务操作建议

**风险识别**：
- 供应商回扣、学术推广费、会议赞助、旅游招待等隐性利益输送
- 代理商/经销商的高额佣金是否实质为商业贿赂
- 特定关系人（配偶、子女）收受财物是否构成"利用职务便利"

**合规建议**：
- 建立礼品/招待上限制度（如单次不超过500元）
- 明确禁止代理商使用任何形式的佣金变相行贿
- 反腐败培训纳入年度必修培训

## 五、出口管制 EAR（美国出口管理条例）

### 5.1 核心框架

EAR（Export Administration Regulations，15 C.F.R. Parts 700-799）由美国商务部工业与安全局（BIS）执行，管制军民两用物项的出口、再出口和转移。

### 5.2 ECCN 编码（Export Control Classification Number）

| 分类 | 说明 | 许可要求 |
|------|------|---------|
| 0Y（弹药类） | 临时分类，需逐案申请 | 严格许可 |
| 1Y/2Y/3Y/4Y | 核/Nuclear类 | 严格许可 |
| 5Y/6Y/7Y/8Y/9Y | 加密/航天/化学等 | 分级许可 |
| EAR99 | 不在控制清单上，但可能受其他管制 | 最宽松 |

**对中国企业的重点**：
- 美国企业向中国出口 EAR 管控物项，需申请出口许可
- 中国企业从美国采购用于再出口，需确保最终用途合规
- **实体清单（Entity List）**：华为等企业几乎无法获得美国技术出口许可

### 5.3 最终用户（End-User）和最终用途（End-Use）管制

**实体清单（Entity List）**：特定中国实体进口美国技术需特别许可

**未经验证清单（UVL）**：BIS 对外国实体的美国物项进口进行验证，列入UVL则引发额外审查

**防止转出口（Diversion）要点**：
- 合同须包含最终用户声明和最终用途声明
- 对代理商/分销商实施监督，防止"老鼠仓"式的转售
- 定期审计终端用户

### 5.4 EAR 合规体系建设

1. ** ECCN 分类筛查**：采购的原材料/设备是否含有美国成分，需确认 ECCN 编码
2. **许可申请**：涉美管控物项的对华出口，需提前申请 BIS 许可
3. **最终用户审查**：建立客户筛查流程（配合 OFAC 筛查）
4. **内部培训**：涉及进出口业务的部门须定期培训
5. **记录保存**：出口文件保存至少5年

### 6.1 商业贿赂主要罪名对照

| 罪名 | 法条 | 主体 | 常见场景 |
|------|------|------|---------|
| 非国家工作人员受贿罪 | 刑法第163条 | 企业员工/管理层 | 采购收回扣、吃差价 |
| 行贿罪 | 刑法第389条 | 任何主体 | 向企业人员行贿 |
| 对非国家工作人员行贿罪 | 刑法第164条 | 任何主体 | 向民营企业人员行贿 |
| 对外国官员行贿罪 | 刑法第164条（FCPA域外） | 在美有业务的企业 | 跨境商业腐败 |
| 串通投标罪 | 刑法第223条 | 投标人 | 围标、陪标 |

### 6.2 风险等级评估矩阵

| 风险维度 | 低风险 | 中风险 | 高风险 |
|---------|--------|--------|--------|
| 招待金额 | 单次<500元 | 500-2000元 | >2000元或现金 |
| 频率 | 偶发 | 季度性 | 月度或更频繁 |

| 地域 | 中国大陆合规企业 | 监管宽松地区 | 高腐败指数国家 |

### 6.3 合规建议

- 对高频供应商实施 **定期轮岗**
- 建立 **礼品招待申报系统**
- 对涉政府业务建立 **政府接触政策**
- 聘请外部机构进行 **反腐败合规审计**

## 七、合规体系建设路径（PDCA循环）

### 7.1 Plan（计划）

- 董事会/管理层承诺，发布合规宣言
- 组建合规委员会，明确汇报线
- 开展全公司合规风险评估（Risk Assessment）
- 制定合规政策和手册

### 7.2 Do（执行）

- 制定各业务线合规操作流程（SOP）
- 开展全员合规培训（含反腐败、反垄断、数据保护）
- 建立违规举报渠道（ hotline + 匿名邮箱）
- 执行合同合规审查

### 7.3 Check（检查）

- 定期内部审计，抽查高风险领域（如销售、采购）
- 合规委员会季度审查
- 外部律所年度评估
- 模拟FCPA审计（针对在美上市或涉美业务企业）

### 7.4 Act（改进）

- 对审计发现制定整改计划
- 更新合规政策和培训内容
- 对违规行为严肃处理（零容忍）
- 定期向董事会汇报合规状况

## 7. FCPA自愿披露减罚机制

> 基于James M. Peck（Cleary Gottlieb Steen & Hamilton）FCPA合规分析。FCPA执法中，自愿披露可以显著降低罚款。

### FCPA自愿披露减罚标准（DOJ指引）

| 合作程度 | 自愿披露 | 及时配合 | 完全合作 | 罚款减罚幅度 |
|----------|---------|---------|---------|------------|
| **最高** | ✅ 24小时内 | ✅ 全面 | ✅ 完全 | 减少50~70% |
| **高** | ✅ | ✅ | ⚠️ 部分 | 减少30~50% |
| **中** | ❌ | ✅ | ✅ | 减少20~30% |
| **低** | ❌ | ⚠️ 部分 | ⚠️ 部分 | 减少0~20% |

### 自愿披露的核心要求

1. **时间要求**：在DOJ发现违法事实之前主动披露（通常为24小时内）
2. **内容要求**：必须披露所有已知的违法行为，不得隐瞒或选择性地披露
3. **持续要求**：在调查期间持续配合，不得有任何妨碍调查的行为

### 不能减轻处罚的因素

即使完全配合，以下情况仍会导致较高处罚：
- 造成实际被害人伤害（如海外行贿导致合同欺诈受害方）
- 恶劣影响（系统性、大规模腐败）
- 持续性违法（多年连续违规）
- 高管/董事直接参与
- 在调查期间妨碍司法

### DOJ罚款计算公式

$$
基础罚款 = 违法所得（ill-gotten gains）× (1 ~ 1.5)
$$

**加重因素（+百分比）**：
- 被害人伤害：+10~20%
- 恶劣影响/系统性：+10~20%
- 持续性违法：+10~20%
- 高管参与：+10~15%
- 妨碍调查：+15~25%

**减轻因素（-百分比）**：
- 自愿披露：-15~30%
- 及时配合：-10~20%
- 完全合作：-10~15%
- 补救措施（合规整改）：-10~20%
- 历史合规记录：-5~10%

$$
最终罚款 = 基础罚款 × (1 + Σ加重因素) × (1 - Σ减轻因素)
$$

**上限**：不超过违法所得的2倍

### 实际案例参考

| 案例 | 公司 | 罚款 | 自愿披露减罚 | 合作情况 |
|------|------|------|------------|---------|
| 摩根大通中国案（2016） | JPM | 2.64亿美元 | 无（未自愿披露） | 部分配合 |
| 中兴通讯（2017） | ZTE | 8.92亿美元 | 否 | 配合 |
| 德勤中国（2023） | Deloitte | 2000万美元 | 是 | 完全配合 |

*增强来源：James M. Peck（Cleary Gottlieb）FCPA合规与执法分析，DOJ官方指引文件*

## 8. DOJ罚款计算公式（FCPA）

> 基于Henry J. Liu（DOJ FCPA执法部门）官方指引文件。系统梳理DOJ对FCPA罚款的计算框架。

### 罚款计算基础

**违法所得（Ill-gotten gains）计算**：

| 违法类型 | 计算方法 | 说明 |
|---------|---------|------|
| 商业贿赂获得的合同收入 | 合同利润金额 | 通常为合同金额的10~30% |
| 通过不正当竞标获得的收入 | 中标金额 - 合规竞争者报价 | 反映非法竞争优势 |
| 其他违法所得 | DOJ酌情确定 | 灵活裁量 |

**基础罚款区间**：
$$
基础罚款 = 违法所得 × (1 ~ 1.5)
$$

DOJ通常取1.0~1.3倍，具体取决于违法持续时间和范围。

### 加重因素（+百分比）

| 加重因素 | 罚款增加幅度 | 说明 |
|---------|-------------|------|
| 被害人伤害 | +10~20% | 造成实际损害，如海外行贿导致合同欺诈 |
| 恶劣影响/系统性 | +10~20% | 大规模/长期系统性腐败 |
| 持续性违法 | +10~20% | 多年连续违规，非单一行为 |
| 高管/董事直接参与 | +10~15% | 管理层知悉或参与 |
| 妨碍调查 | +15~25% | 销毁文件、拒绝配合、员工威胁等 |
| 在执法程序中撒谎 | +5~10% | 作伪证或提供虚假文件 |

### 减轻因素（-百分比）

| 减轻因素 | 罚款减少幅度 | 说明 |
|---------|-------------|------|
| 自愿披露 | -15~30% | 违法发现后主动向DOJ披露 |
| 及时配合 | -10~20% | 及时提供相关文件，安排人员配合访谈 |
| 完全合作 | -10~15% | 全面配合调查，无隐瞒 |
| 补救措施 | -10~20% | 已实施合规整改、解雇涉事人员等 |
| 历史合规记录 | -5~10% | 有合规培训、内部审计历史 |
| 合规体系建设 | -10~20% | 已有或已承诺建设合规体系 |

### 最终罚款计算公式

$$
最终罚款 = 基础罚款 × (1 + Σ加重因素) × (1 - Σ减轻因素)
$$

**上限**：不超过违法所得的**2倍**

**下限**：通常不低于违法所得的**0.5倍**

### DOJ罚款计算示例

**案例背景**：
- 公司通过海外行贿获得1000万美元合同利润
- 违法持续3年（持续性违法：+15%）
- 高管参与（+10%）
- 自愿披露（-20%）
- 完全配合（-10%）
- 已实施合规整改（-15%）

**计算**：
```
基础罚款 = 1000万 × 1.2 = 1200万
Σ加重因素 = 15% + 10% = 25%
Σ减轻因素 = 20% + 10% + 15% = 45%
最终罚款 = 1200万 × (1 + 25%) × (1 - 45%)
         = 1200万 × 1.25 × 0.55
         = 825万
```

**上限检查**：825万 < 1000万×2 = 2000万 ✓

## 9. 刑事合规不起诉操作流程（中国）

> 基于刘艳（金杜律师事务所）企业合规培训材料。中国版合规不起诉制度于2020年起试点，适用于企业重大单位犯罪。

### 适用条件

1. **案件类型**：公司涉嫌重大单位犯罪（如欺诈贿赂、环境污染、安全生产事故等）
2. **责任人条件**：直接责任人认罪认罚
3. **公司条件**：具备整改意愿和合规整改条件
4. **检察官裁量**：检察机关认为不起诉有利于社会公共利益

### 操作流程（6步）

| 步骤 | 内容 | 时长 | 执行主体 |
|------|------|------|---------|
| **Step 1** | 检察机关发出检察建议 | ~10个工作日 | 检察机关 |
| **Step 2** | 企业提交合规整改计划 | ~30日 | 企业 |
| **Step 3** | 企业启动合规整改（PDCA） | 3~6个月 | 企业 |
| **Step 4** | 第三方评估机构评估 | ~1个月 | 第三方机构 |
| **Step 5** | 检察官验收+听证 | ~1个月 | 检察机关 |
| **Step 6** | 检察机关作出相对不起诉决定 | — | 检察机关 |

**总时长**：约90~180天

### PDCA合规体系建设框架

- **Plan（计划）**：识别合规风险 → 制定合规制度 → 明确责任人
- **Do（执行）**：全员合规培训 → 流程嵌入 → 举报机制建立
- **Check（检查）**：定期合规审计 → 违规行为处理 → 整改跟踪
- **Act（改进）**：制度更新 → 风险再评估 → 文化强化

### 合规整改核心领域

| 领域 | 典型整改措施 |
|------|-------------|
| 反商业贿赂 | 礼品/招待政策、第三方尽调、付款合规 |
| 数据合规 | 个人信息保护、跨境传输合规、数据分类分级 |
| 安全生产 | 安全管理制度、隐患排查、应急预案 |
| 环境保护 | 排污许可、监测体系、危废处理 |

### 黄金标准（DOJ合规指引参考）

检察机关评估合规体系时参考：
- 合规体系和政策的充分性
- 高层领导对合规的承诺
- 风险评估和应对
- 员工培训和沟通
- 报告和惩戒机制
- 第三方管理
- 持续改进机制
