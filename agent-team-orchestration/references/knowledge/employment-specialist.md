# 劳动法专家知识库

> 企业法务劳动法专家（employment-specialist）的本地知识手册。用于劳动关系处理和劳动争议解决。

---

## 1. 劳动合同法核心条款

### 劳动合同解除与终止

**用人单位合法解除（无补偿）**（《劳动合同法》第39条）：
1. 试用期内被证明不符合录用条件
2. 严重违反用人单位规章制度
3. 严重失职，营私舞弊，给用人单位造成重大损害
4. 同时与其他用人单位建立劳动关系，对完成本单位工作任务造成严重影响，或经用人单位提出，拒不改正
5. 以欺诈、胁迫或乘人之危，使对方在违背真实意思的情况下订立或变更劳动合同
6. 被依法追究刑事责任

**经济性裁员**（《劳动合同法》第41条）：
- 需要裁减人员20人以上或占企业职工总数10%以上
- 需提前30日向工会或全体职工说明情况
- 向劳动行政部门报告

### 经济补偿计算

$$经济补偿 = 月工资 \times 补偿月数$$

| 工作年限 | 补偿月数 |
|----------|----------|
| 6个月以下 | 0.5个月 |
| 6个月~1年 | 1个月 |
| 每满1年 | 1个月 |
| >12年 | 最高12个月 |

**月工资**：离职前12个月平均工资（包含奖金、津贴等）。

## 2. 劳动争议处理流程

```
协商 → 调解 → 劳动仲裁 → 诉讼（不服仲裁）
```

### 劳动仲裁时效

- **一般时效**：从知道或应当知道权利被侵害之日起**1年**
- 劳动关系存续期间因拖欠劳动报酬发生争议的：**不受1年限制**，劳动关系终止后1年内提出

### 仲裁申请材料

1. 仲裁申请书（2份）
2. 身份证复印件
3. 劳动合同/社保缴费记录/工资流水（证据）
4. 用人单位工商信息（国家企业信用信息公示系统查询）

## 3. 规章制度有效性审查

### 有效规章制度三要素（司法解释一）

1. **制定程序民主**：经职工代表大会或全体职工讨论，提出方案和意见，与工会或职工代表平等协商确定
2. **内容合法**：不违反法律、法规的规定
3. **公示告知**：向劳动者公示或告知（签字确认/培训记录/手册发放）

### 常见无效条款

- 末位淘汰制（"员工考核垫底即解除"）：违反《劳动合同法》第39条
- 入职X年内不得生育：违反《就业促进法》《妇女权益保障法》
- 加班不支付加班费：违反《劳动法》第44条

## 4. 社保合规要点

### 强制缴纳范围

| 险种 | 用人单位缴费比例（参考） | 个人缴费比例 |
|------|----------------------|------------|
| 养老保险 | 16% | 8% |
| 医疗保险 | 8.5~10% | 2% |
| 失业保险 | 0.7% | 0.3% |
| 工伤保险 | 0.2~1.9%（按行业） | 不缴费 |
| 生育保险 | 0.8%（并入医疗） | 不缴费 |

### 公积金缴纳

- 缴存比例：5%~12%（企业自选）
- 缴存基数：上一年度月平均工资（上下限为当地社平工资的3倍）

### 不缴社保的法律后果

1. **员工可主张被迫离职**：要求支付经济补偿（工作一年补偿一个月工资）
2. **行政部门责令改正**：逾期不缴的，处欠缴数额1~3倍罚款
3. **员工可要求补缴**：劳动监察或仲裁要求补缴
