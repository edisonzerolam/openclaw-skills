# 资金追踪师知识库

> 资金追踪师（money-flow-tracker）的本地知识手册。用于资金面分析时的指标/数据/阈值引用。

---

## 1. 北向资金（沪深港通）

### 数据含义

北向资金 = 通过沪港通和深港通从香港流入A股的国际资金。

| 指标 | 含义 | 判断标准 |
|------|------|----------|
| **当日净流入** | 当日买入额 - 当日卖出额 | 正=买入，负=卖出 |
| **累计净流入** | 历史累计净买入总额 | 趋势性增加=持续看好 |
| **当日成交买入占比** | 买入额/沪深港通总成交额 | >30%=主动买入意愿强 |

### 净流入分级

| 当日净流入 | 信号强度 | 市场含义 |
|------------|----------|----------|
| >50亿 | 强烈买入 | 国际资金大幅建仓 |
| 20~50亿 | 中等买入 | 积极信号 |
| 5~20亿 | 轻微买入 | 温和看多 |
| -5~5亿 | 持平 | 无明显方向 |
| -20~-5亿 | 轻微卖出 | 温和看空 |
| -50~-20亿 | 中等卖出 | 明显减仓 |
| <-50亿 | 强烈卖出 | 国际资金大幅抛售 |

### 北向资金持仓结构

- **前5大持仓行业**：食品饮料、电力设备、医药生物、银行、电子
- **持仓集中度**：前10大个股占持仓总市值的40~50%
- **加仓/减仓信号**：周度环比变化 >5% 为明显异动

## 2. 融资融券余额

### 核心指标

| 指标 | 计算 | 市场含义 |
|------|------|----------|
| **融资余额** | 投资者向券商借入资金买入股票的未还金额 | 反映杠杆做多情绪 |
| **融券余额** | 投资者向券商借入股票卖出的未还金额 | 反映杠杆做空情绪 |
| **融资融券比** | 融资余额 / 融券余额 | >10x=多头情绪占主导 |

### 融资余额与市场关系

**融资余额增速** = （本期融资余额 - 上期融资余额）/ 上期融资余额

| 融资余额增速 | 市场含义 |
|------------|----------|
| 周度增速 >5% | 杠杆资金加速入场，市场热度高 |
| 周度增速 2~5% | 温和加杠杆，正常多头 |
| 周度增速 -2~2% | 持平，中性 |
| 周度增速 -5~-2% | 去杠杆，谨慎 |
| 周度增速 <-5% | 杠杆资金加速离场，风险偏好下降 |

### 融资余额绝对值历史分位

- 融资余额处于近5年80百分位以上 → 杠杆资金过热，风险累积
- 融资余额处于近5年20百分位以下 → 杠杆资金过冷，情绪底部

## 3. 主力资金净流入

### 单笔成交资金分类标准（通达信标准）

| 分类 | 成交额区间 |
|------|-----------|
| **超大单** | >100万元/笔 |
| **大单** | 20~100万元/笔 |
| **中单** | 5~20万元/笔 |
| **小单** | <5万元/笔 |

### 主力资金计算

$$
主力净流入 = 超大单净买入额 + 大单净买入额
$$

$$
个股主力净流入 = 超大单买入 - 超大单卖出 + 大单买入 - 大单卖出
$$

### 主力净流入分级

| 主力净流入（万元） | 信号 |
|------------------|------|
| >5000 | 超强买入信号 |
| 2000~5000 | 强买入 |
| 500~2000 | 温和买入 |
| -500~500 | 中性 |
| -2000~-500 | 温和卖出 |
| -5000~-2000 | 强卖出 |
| <-5000 | 超强卖出 |

**注意**：主力资金数据在日末（T+1）才能获取，无盘中实时数据。

## 4. 大盘资金流向（日度）

### 全市场资金流向计算

$$
全市场主力净流入 = \sum_{i=1}^{N} 个股主力净流入_i
$$

### 资金流向与指数关系

| 情境 | 沪深300走势 | 主力净流入 | 含义 |
|------|------------|-----------|------|
| **情景A** | 上涨 | 正流入 | 资金驱动型上涨，健康 |
| **情景B** | 上涨 | 负流入 | 散户拉抬，主力出货 |
| **情景C** | 下跌 | 正流入 | 主力逆势吸筹 |
| **情景D** | 下跌 | 负流出 | 主动卖出，压力较大 |

**情景A和C** 最为健康；**情景B** 需要警惕主力出货；**情景D** 需等待情绪稳定。

## 5. 龙虎榜解读

### 什么是龙虎榜

龙虎榜 = 上交所/深交所每日公布的异常波动股票交易席位信息。

### 上榜条件（满足任一）

1. 当日收盘涨跌幅偏离值 ±7%
2. 当日振幅 15%
3. 当日换手率 20%
4. 连续3个交易日涨跌幅偏离值累计 20%

### 关键数据字段

| 字段 | 含义 |
|------|------|
| **机构席位买入** | 机构专用账户买入金额 |
| **机构席位卖出** | 机构专用账户卖出金额 |
| **营业部买入** | 游资/散户聚集席位 |
| **营业部卖出** | 游资/散户卖出席位 |

### 龙虎榜信号判断

| 机构买入/营业部买入比 | 信号 |
|---------------------|------|
| >1.5 | 机构主导，看涨信号强 |
| 0.8~1.5 | 机构参与度较高 |
| 0.3~0.8 | 游资主导，炒作性质 |
| <0.3 | 纯游资炒作，高风险 |

### 营业部席位常见游资

- **著名游资席位**：华泰证券深圳益田路、中信证券量化等
- **连续上榜意义**：个股有明确主力资金持续运作，需关注后续走势
