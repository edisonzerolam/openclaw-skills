# 判例分析师 — 类案检索与分析知识库

## 一、类案检索方法

### 1.1 检索维度体系

| 检索维度 | 说明 | 检索优先级 |
|----------|------|------------|
| **案由** | 案件类型的规范化分类 | ★★★ 首选 |
| **当事人** | 原告/被告/第三人的名称 | ★★☆ 辅助 |
| **争议焦点** | 案件核心法律问题 | ★★★ 核心 |
| **法律关系** | 合同/侵权/物权等法律关系类型 | ★★☆ 辅助 |
| **诉讼请求** | 原告的具体诉请 | ★☆☆ 参考 |
| **法院认为** | 法官论证中的法律适用表述 | ★★☆ 高频 |

### 1.2 案由导航检索

依据最高人民法院《民事案件案由规定》（法〔2020〕347号），通过案由树逐级定位：

```
示例路径（买卖合同纠纷）：
第四级案由：买卖合同纠纷
    ↓
可检到：
- 分期付款买卖合同纠纷
- 凭样品买卖合同纠纷
- 试用买卖合同纠纷
- 互易合同纠纷
```

### 1.3 关键词检索策略

```
检索公式：
[法律问题] + [关键事实] + [裁判结果]

示例1（建设工程质量纠纷）：
"建设工程 质量 鉴定 合格 违约金"

示例2（竞业限制纠纷）：
"竞业限制 补偿金 继续履行 竞争企业"

示例3（网络服务合同纠纷）：
"网络服务 用户协议 格式条款 有效"
```

### 1.4 类案相似度判断标准

| 相似度层级 | 判定标准 | 适用场景 |
|------------|----------|----------|
| **高度相似** | 案由相同+当事人类型相同+争议焦点相同+关键事实近似 | 直接参照 |
| **中度相似** | 案由相同+法律关系类型相同+核心问题近似 | 参考裁判思路 |
| **低度相似** | 案由相同+仅部分事实相似 | 借鉴论证逻辑 |

## 二、判例分析方法

### 2.1 事实相似度分析框架

```
分析维度（逐项比对）：

1. 主体相似性
   □ 相同行业/领域
   □ 相同企业类型（上市公司/中小企业/个人）
   □ 相同身份（消费者/劳动者/经营者）

2. 行为相似性
   □ 相同行为模式
   □ 相同履行方式
   □ 相同违约/侵权形态

3. 结果相似性
   □ 相同损害类型
   □ 相同损失范围
   □ 相同责任形式

4. 时间地点相似性
   □ 相近时期（法律变化影响）
   □ 相同地域（司法实践差异）
```

### 2.2 法律适用分析

```
分析步骤：

Step 1：法律依据梳理
   - 引用法条是否一致
   - 是否引用司法解释
   - 是否引用指导性案例

Step 2：法律解释方法
   - 文义解释路径
   - 体系解释路径
   - 目的解释路径

Step 3：裁量标准
   - 自由裁量幅度（如违约金调整比例）
   - 损害赔偿计算方法
   - 因果关系认定标准

Step 4：证据认定规则
   - 举证责任分配
   - 证明力大小判断
   - 证据链构建要求
```

### 2.3 判例结构解析

```
判例标准结构：

1. 案件基本信息
   - 案号
   - 审理法院/层级
   - 裁判日期
   - 案件类型

2. 案件事实（经审理查明）
   - 当事人关系
   - 事件经过
   - 争议焦点

3. 原告诉请
   - 诉讼请求清单
   - 事实与理由

4. 被告答辩
   - 抗辩理由
   - 事实与理由

5. 法院认为（核心）
   - 争议焦点归纳
   - 各焦点逐一分析
   - 法律适用论证
   - 裁判结果推导

6. 裁判主文
   - 判决/裁定主文
   - 案件受理费负担
   - 上诉权利告知
```

## 三、裁判倾向分析

### 3.1 常见裁判规律

| 纠纷类型 | 常见裁判倾向 | 影响因子 |
|----------|--------------|----------|
| **合同解除** | 违约方解除权严格限制 | 合同僵局/长期性/不可替代性 |
| **违约金调整** | 以实际损失为基础，30%为参考线 | 合同标的/履行程度/过错 |
| **格式条款** | 倾向于认定无效或做出不利于提供方解释 | 提示义务/合理性/公平性 |
| **消费者维权** | 支持退一赔三/十，惩罚性赔偿倾向明显 | 欺诈认定/明知要件 |
| **劳动争议** | 程序前置，仲裁时效严格把握 | 劳动关系确认/时效起算点 |

### 3.2 违约金调整裁审标准

**法条依据**：
- 《民法典》第585条（当事人可以约定违约金）
- 《民法典合同编解释》第65条（违约金过高调整）

**调整标准**：

```
过高认定参考标准（司法实践）：
- 约定违约金超过造成损失的30%时，一般认定为"过高"
- 法院有权依据当事人请求予以适当减少

损失计算方法：
1. 实际损失（证明责任：守约方）
2. 可得利益损失（综合考虑可预见规则/减损规则/损益相抵）
3. 举证困难时的酌定

调整幅度参考：
- 金融借款类：一般不超过LPR的4倍
- 民间借贷类：不超过LPR的4倍（2020年8月20日后）
- 一般商事合同：30%参考线，但有酌情调整空间
- 消费者合同：倾向保护消费者，适当调低
```

### 3.3 举证责任分配规则

```
"谁主张，谁举证"原则（《民事诉讼法》第67条）

举证责任分配标准：
1. 基本原则：主张积极事实的当事人承担举证
2. 消极事实免责：主张消极事实（未违约/未侵权）方一般不承担举证

特殊举证责任（法律另有规定）：
- 举证责任倒置：环境污染/产品责任/医疗损害/劳动安全
- 举证责任转移：对方持有证据而不提供的

证明标准：
- 民事一般标准：高度可能性（优势证据）
- 特殊标准：排除合理怀疑（欺诈/胁迫/恶意）
- 程序事项：较大可能性
```

## 四、指导性案例效力

### 4.1 指导性案例地位

```
法律依据：
《最高人民法院关于案例指导工作的规定》（法发〔2010〕51号）

效力定位：
- 各级人民法院审理类似案件时应当参照
- 不是正式的法律渊源
- 具有事实上的约束力（实践中基本参照）

编号规则：
"最高人民法院指导性案例第XX号"
```

### 4.2 指导性案例检索

```
检索路径：
1. 最高人民法院官网→案例指导
2. 中国指导性案例网
3. 北大法宝→指导性案例数据库
4. 裁判文书网→指导性案例筛选

筛选方法：
- 按案由筛选
- 按关键词筛选
- 按发布年份筛选
- 按裁判法院层级筛选
```

### 4.3 参照适用规则

```
参照步骤：
1. 识别类似案件：
   - 案件基本相同（案由/事实/法律关系）
   - 指导性案例具有指导意义

2. 区分适用：
   - 事实上的差异是否实质影响裁判
   - 指导性案例的事实要件是否满足

3. 区分理由：
   - 不同点应明确说明
   - 不同点对法律适用影响的论证

4. 不参照时的程序：
   - 应当报送本院审判委员会讨论
   - 不参照无需说明理由（惯例做法）
```
