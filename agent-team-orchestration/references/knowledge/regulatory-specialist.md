# 监管合规专家知识库

> 企业法务监管合规专家（regulatory-specialist）的本地知识手册。用于行业监管合规评估和合规体系建设。

---

## 1. 主要行业监管框架（A股上市公司）

| 行业 | 主要监管部门 | 核心法规 |
|------|------------|----------|
| **金融** | 银保监会/证监会/央行 | 银行业监督管理法、证券法、基金法 |
| **医药** | 国家药监局/卫健委 | 药品管理法、医疗器械监督管理条例 |
| **食品** | 市场监管总局 | 食品安全法 |
| **教育** | 教育部/省级教育厅 | 教育法、民促法（营利性/非营利性） |
| **互联网** | 网信办/工信部/市场监管总局 | 网络安全法、数据安全法、个人信息保护法 |
| **房地产** | 住建部/自然资源部 | 城市房地产管理法 |
| **能源** | 国家能源局/发改委 | 能源法、电力法 |

## 2. 合规管理体系（COCO/GRC框架）

### 合规管理基本架构

```
合规治理层
├── 董事会/监事会
├── 合规负责人（首席合规官）
└── 合规管理委员会

合规执行层
├── 各业务部门（合规第一责任）
├── 合规管理部门（统筹协调）
└── 内部审计部门（监督评价）

合规支撑层
├── 合规管理制度
├── 合规风险评估
├── 合规培训
└── 合规举报机制
```

### 合规风险评估方法

**风险等级 = 发生可能性 × 影响程度**

| 可能性\影响 | 低影响 | 中影响 | 高影响 |
|------------|--------|--------|--------|
| 高可能 | 中风险 | 高风险 | 极高风险 |
| 中可能 | 低风险 | 中风险 | 高风险 |
| 低可能 | 低风险 | 低风险 | 中风险 |

## 3. 监管检查应对流程

### 日常合规监控

| 监控项 | 频率 | 负责部门 |
|--------|------|----------|
| 法规更新跟踪 | 月度 | 合规部 |
| 业务合规自查 | 季度 | 业务部门 |
| 合规培训 | 半年度 | 人力+合规 |
| 合规审计 | 年度 | 内部审计 |

### 监管检查应对步骤

**1. 收到检查通知**：
- 确认检查范围/依据/时间
- 成立迎检工作组
- 准备证照/制度/业务记录

**2. 现场检查配合**：
- 安排专人接待
- 如实提供资料（不得拒绝/隐瞒）
- 对询问做好记录

**3. 检查后整改**：
- 收到整改通知书后30日内提交整改方案
- 按计划落实整改
- 保留整改证据

## 4. 合规报告标准

### 合规报告类型

| 报告类型 | 提交对象 | 提交时限 |
|----------|----------|----------|
| 定期合规报告 | 董事会/监管部门 | 年度/季度 |
| 重大违规报告 | 董事会/监管部门 | 5个工作日内 |
| 专项合规报告 | 按需 | 按需 |

### 重大违规事件判断标准

1. 涉及罚款金额>100万
2. 被采取行政强制措施
3. 被责令停产停业
4. 累计违规事项>3项
5. 可能影响公司上市地位的事件
