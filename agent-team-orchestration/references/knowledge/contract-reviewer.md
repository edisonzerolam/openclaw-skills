# 合同审查师知识库

> 合同审查师（contract-reviewer）的本地知识手册。用于合同审查时的法条/清单/标准引用。

---

## 1. 合同条款审查清单

### 核心条款（五要件）

| 条款 | 审查要点 |
|------|----------|
| **当事人主体** | 名称/统一社会信用代码是否准确；是否为适格主体；是否有限制能力（如未成年人、精神病人） |
| **标的** | 标的物/服务/权利是否具体明确；数量是否清晰；质量标准是否可量化 |
| **价款/报酬** | 金额是否确定；支付方式/时间是否清晰；货币种类是否明确 |
| **履行** | 履行时间/地点/方式是否明确；履行条件是否合理 |
| **违约责任** | 违约金是否合理（过高可申请司法调整）；损失赔偿范围是否明确；不可抗力条款是否合理 |

### 格式条款审查

依据《民法典》第496条：

- 是否以"合理方式"提请对方注意（加粗/标红/单独签署）
- 是否存在加重对方责任、排除对方主要权利的条款（无效）
- 免责条款是否满足《民法典》第506条（造成对方人身损害/故意或重大过失）

### 合同解除条件审查

**法定解除权**（《民法典》第563条）：
1. 不可抗力导致目的无法实现
2. 履行期限届满前，一方明确表示或以行为表明不履行主要债务
3. 一方迟延履行主要债务，经催告后在合理期限内仍未履行
4. 一方迟延履行或其他违约致使不能实现合同目的
5. 法律规定的其他情形

**约定解除**：是否约定了明确的解除条件和解约程序。

## 2. 常见风险条款识别

### 高风险条款清单

1. **管辖权条款**：约定对自己不利的法院或仲裁机构
2. **送达条款**：仅写地址未注明"包括电子送达"→ 法律文书送达困难
3. **税费条款**：约定"税费由乙方承担"→ 可能被认定为转嫁本应由甲方承担的税费
4. **自动续期条款**：无明确续期条件，自动续期可能锁定长期义务
5. **竞业限制过宽**：范围过宽/期限过长/补偿不足 → 无效
6. **单方变更权**：赋予一方单方调整合同重大条款的权利 → 可能无效

### "霸王条款"识别（无效条款）

| 条款 | 法律后果 |
|------|----------|
| 造成对方人身伤害免责 | 无效 |
| 因故意/重大过失造成对方财产损失免责 | 无效 |
| 限制对方索赔权 | 无效 |
| 单方决定转移义务或权利 | 无效 |
| 排除对方解约权 | 无效 |

## 3. 违约金调整规则

### 司法调整标准

依据《民法典》第585条：

**约定违约金过高**：当事人可申请法院或仲裁机构予以减少

- **调整标准**：实际损失的130%为上限
- **实际损失**：包括合同履行后可以获得的利益

**计算公式**：
$$
违约金上限 = \max(实际损失, 预期利益损失) \times 130\%
$$

### 法院实践中的常见认定

| 违约金/实际损失比 | 法院认定 |
|------------------|----------|
| < 30% | 可能被认定为过低而调高 |
| 30~100% | 合理区间 |
| 100~150% | 法院可能酌情调减 |
| > 150% | 通常认定为过高，会被调减 |

### 过低违约金的调整

如果违约金低于实际损失：
- 依据《民法典》第585条，可申请增加
- 调整后不超过实际损失

## 4. 合同审核评分卡

### 合同质量评分体系

| 维度 | 分值 | 评估标准 |
|------|------|----------|
| **主体适格性** | 20分 | 当事人信息准确、主体合法 |
| **条款完整性** | 20分 | 五要件齐全 |
| **权利义务对等性** | 20分 | 双方权利义务基本对等 |
| **风险条款数量** | 20分 | 无高风险条款 |
| **可执行性** | 20分 | 条款具体可操作，无歧义 |

**总分**：
- 90~100：优质合同
- 70~89：良好，轻微修改
- 50~69：需重大修改
- <50：风险过高，建议重新谈判或放弃
