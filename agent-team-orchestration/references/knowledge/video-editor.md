# 视频剪辑师知识库

> 视频剪辑师（video-editor）的本地知识手册。用于视频剪辑时的节奏控制/转场/规格参数引用。

---

## 1. 剪辑节奏控制

### 时长与注意力模型

| 视频时长 | 适用场景 | 注意力极限 |
|----------|----------|------------|
| **15秒内** | 抖音信息流、悬念型内容 | 全程高能，每3秒一个刺激点 |
| **30~60秒** | 知识科普、观点输出 | 3~5个内容模块 |
| **1~3分钟** | 深度讲解、测评 | 开头5秒+核心内容+结尾call-to-action |
| **5~10分钟** | 教程、vlog | 分段标题+章节导航 |
| **10分钟以上** | 系列内容、纪录片 | 多章节结构，节奏起伏设计 |

### 剪辑节奏公式

$$
剪辑密度 = \frac{镜头总数}{视频总时长（秒）}
$$

| 剪辑密度 | 节奏评价 | 适用类型 |
|----------|----------|----------|
| < 0.5 | 慢节奏，大画面 | 演讲、纪录片、访谈 |
| 0.5~1.5 | 中等节奏 | 知识科普、vlog |
| 1.5~3.0 | 快节奏 | 娱乐、卡点视频 |
| > 3.0 | 极快节奏 | 混剪、快剪 |

## 2. 转场类型及使用场景

### 常用转场分类

| 转场类型 | 描述 | 使用场景 |
|----------|------|----------|
| **硬切（Cut）** | 无过渡直接切换 | 日常记录、对话、强调对比 |
| **叠化（Cross Dissolve）** | A画面逐渐被B替代 | 时间流逝、场景转换、柔和过渡 |
| **闪白（Flash）** | 短促白帧/闪光线 | 节奏高潮点、转场、强调 |
| **模糊（Blur）** | 画面模糊过渡 | 静动态切换、情绪转场 |
| **滑（Slide）** | 画面横向滑入/出 | 列表展示、时间线、对比 |
| **缩放（Zoom）** | 画面放大/缩小过渡 | 强调、转场 |
| **黑场（Black Fade）** | 纯黑过渡 | 章节分割、时间跳跃 |
| **速度变化** | 慢动作/快进 | 情绪表达、时间压缩 |

### 转场选择原则

1. **同场景内**：用硬切，保持流畅
2. **场景转换**：叠化或黑场，不要用花哨转场
3. **节奏高潮**：闪白、速度变化
4. **信息对比**：滑轨/缩放
5. **禁止**：滥用"旋转""模糊圈"等复杂转场（破坏观看体验）

## 3. 字幕与音效规范

### 字幕规范

**中文字幕**：
- 字体：黑体/思源黑体，不使用宋体（宋体在视频中不清晰）
- 大小：画面高度的1/15~1/20
- 颜色：白色加黑色描边（#FFFFFF + 黑边2px）
- 位置：画面底部1/3区域，居中
- 行间距：字号的1.2倍
- 每行字数：不超过15字

**英文字幕**：
- 字体：Arial Bold 或 Helvetica
- 大小：不低于720p画面的1/30
- 颜色：白色或黄色

### 音效使用原则

| 音效类型 | 使用场景 | 版权注意 |
|----------|----------|----------|
| **背景音乐（BGM）** | 全程背景氛围 | 必须有版权授权 |
| **音效（Sound FX）** | 关键动作配合 | 可用免版权音效库 |
| **转场音效** | 转场时短促音效 | 短促，版权宽松 |
| **人声配音** | 后期配音需录音质量 | 环境噪音<-40dB |
