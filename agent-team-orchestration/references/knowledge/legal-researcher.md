## 一、中国法律体系层级

### 1.1 效力层级体系（上位法优于下位法）

```
┌─────────────────────────────────────────┐
│           宪  法（最高效力）              │  母法，国家根本法
│        《中华人民共和国宪法》            │
└─────────────────────────────────────────┘
                    ↑
┌─────────────────────────────────────────┐
│            全人制定法律                  │  基本法律/普通法律
│  基本法律：刑法/民法典/诉讼法/组织法    │
│  普通法律：合同法（已废止）→民法典合同编│
└─────────────────────────────────────────┘
                    ↑
┌─────────────────────────────────────────┐
│         国务院行政法规                   │  总理签署国务院令
│    《物业管理条例》《劳动合同法实施条例》 │
└─────────────────────────────────────────┘
                    ↑
┌─────────────────────────────────────────┐
│     国务院部门规章（部委规章）           │  部/委/署/局令
│      《网络支付服务管理办法》等          │
└─────────────────────────────────────────┘
                    ↑
┌─────────────────────────────────────────┐
│         地方性法规/地方政府规章          │  省/设区的市人大
│      《上海市共有产权保障房管理条例》    │
└─────────────────────────────────────────┘
```

## 二、常用法律数据库检索方法

### 2.1 北大法宝（最全面法律数据库）

**访问方式**：https://www.pkulaw.com

**检索策略**：

```
1. 法规检索：
   - 输入条文关键词：如"不安抗辩权 第527条"
   - 限定效力状态：现行/已废止/尚未施行
   - 限定发布机关：全国人大/国务院/部委

2. 案例检索：
   - 案由导航：左侧案由树逐级选择
   - 关键词检索："法院认为"部分的规范表达
   - 限定法院层级：最高院/高院/中院/基层院
   - 限定地域：中院名称如"上海市第一中级人民法院"

3. 专题检索：
   - 实务指引/审判参考/公报案例
   - 法律解读/立法评估
```

### 2.2 裁判文书网（最高人民法院官方）

**访问方式**：https://wenshu.court.gov.cn

**检索策略**：

```
1. 高级检索字段：
   - 当事人信息（当事人名称精确匹配）
   - 案号（如"（2021）沪01民终1234号"）
   - 案由分类
   - 审理法院/法官
   - 文书类型（判决书/裁定书/调解书）

2. 检索技巧：
   - 使用引号精确匹配当事人
   - OR逻辑组合多个关键词
   - "本院认为"字段限定法律适用表述
   - 限定文书类型筛选

3. 局限：
   - 公开率约70-80%，部分案件文书不上网
   - 迟延公开（裁判后数月才上网）
   - 涉及隐私案件依申请不上网
```

## 三、法律条文引用规范

### 3.1 引用格式标准

```
引用层级（由大到小）：
[国家] [机关] [文件名称] [条款编号]

示例规范引用：
- 法律：《中华人民共和国民法典》第五百二十七条
- 行政法规：《物业管理条例》第三十五条
- 司法解释：《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》第十条
- 地方性法规：《上海市消费者权益保护条例》第十五条
```

### 3.2 条款编号规范

```
编→章→节→条→款→项→目
民法典
  ↓
第三编 合同
  ↓
第一章 合同的订立
  ↓
第四百六十九条 【合同订立形式】
  ↓
第一款（法条原文第一句）
  ↓
第（一）项（条下序号）
  ↓
1.（目下序号）
```

**引用示例**：
- 条：`《民法典》第四百六十九条`
- 款：`《民法典》第四百六十九条第一款`
- 项：`《民法典》第四百六十九条第一款第（一）项`
- 目：`《民法典》第四百六十九条第一款第（一）项第1目`

### 3.3 司法解释引用规则

```
引用格式：
《最高人民法院关于适用〈中华人民共和国民法典〉合同编通则若干问题的解释》
（法释〔2023〕13号）

简化引用方式（在首次完整引用后）：
《民法典合同编通则解释》第X条
```

### 3.4 案例引用格式

```
指导性案例（最高人民法院发布）：
"最高人民法院指导性案例XX号"

公报案息（最高人民法院公报）：
"最高人民法院公报20XX年第X期"

典型案例（各部委/各高院发布）：
"最高人民法院发布第二批人民法院贯彻实施民法典典型案例"
```

## 四、法律解释方法

### 4.1 文义解释（字面解释）

**规则**：按条文文字通常含义进行解释

**适用场景**：
- 条文含义清晰、无争议时优先采用
- 不得超越文义可能范围

**局限**：
- 存在多义性时需结合其他方法
- 文义可能与立法原意不符

### 4.2 体系解释（系统解释）

**规则**：将条文置于整个法律体系中考量，维持内部逻辑一致

**具体方法**：
```
1. 章节关联：同一章节条文相互参照
2. 编章贯通：总则与分则相互印证
3. 特别法优先：特别规定优先于一般规定
4. 新法优于旧法：后法优先于前法

示例：
解释"合同解除"时，应联系：
- 合同编通则中的合同解除一般规定
- 合同编典型合同中各类型合同解除的特别规定
- 侵权编中与合同解除竞合的规定
```

### 4.3 目的解释

**规则**：根据法律规范的立法目的进行解释

**操作步骤**：

```
1. 立法目的大小之分：
   - 根本目的（整个法律/编章的立法宗旨）
   - 具体目的（具体条文的规制意旨）

2. 民法典基本立法目的（《民法典》第一条）：
   "为了保护民事主体的合法权益，调整民事关系，维护社会和经济秩序，适应中国特色社会主义发展要求，弘扬社会主义核心价值观，根据宪法，结合我国实际情况，制定本法。"

3. 特别法目的：
   - 《劳动合同法》：保护劳动者合法权益
   - 《消费者权益保护法》：保护消费者合法权益
   - 《环境保护法》：保护和改善环境
```

### 4.4 历史解释（立法史解释）

**规则**：考察立法过程中的历史资料确定规范含义

**资料来源**：
- 法律草案、说明、审议报告
- 座谈会、论证会记录
- 修正案修改说明
- 专家建议稿与立法理由

**注意**：历史解释是辅助方法，不能替代文义解释

### 4.5 解释方法的优先顺序

```
常规顺序（法律无漏洞时）：
文义解释 → 体系解释 → 目的解释

特殊情形（出现歧义/漏洞时）：
文义解释 → 体系解释 → 目的解释 → 历史解释

禁止类推：民事法律无明文规定时，向下位阶法（行政法规/规章）寻找裁判依据时，类推适用应严格限制
```
