# 分诊律师知识库

> 企业法务分诊律师（triage-lawyer）的本地知识手册。用于企业法律问题分类和优先级判断。

---

## 1. 企业法律问题分类体系

### 六大领域

| 领域 | 核心问题 | 紧迫性 | 处理难度 |
|------|----------|--------|----------|
| **合同法务** | 合同审查/谈判/纠纷 | 中高 | 中 |
| **劳动法务** | 劳动合同/劳动关系/劳动争议 | 高（群体性） | 中 |
| **知识产权** | 专利/商标/著作权侵权 | 中低（有时限） | 高 |
| **监管合规** | 行业许可/行政处罚/监管检查 | 高（有时限） | 高 |
| **公司治理** | 股权/控制权/治理结构 | 中 | 高 |
| **并购重组** | 尽职调查/交易结构/审批 | 中 | 极高 |

### 问题优先级判断

| 优先级 | 场景 | 响应时限 |
|--------|------|----------|
| **P0（紧急）** | 监管介入/查封/员工集体仲裁/重大合同违约 | 24小时内 |
| **P1（重要）** | 合同纠纷诉讼/知识产权侵权/劳动仲裁 | 3个工作日内 |
| **P2（常规）** | 合同审查/合规评估/制度完善 | 1周内 |
| **P3（例行）** | 法律咨询/培训/合同模板更新 | 2周内 |

## 2. 分诊路径决策树

### 第一层判断：法律性质

```
收到问题 → 是涉诉还是非诉？
├── 涉诉 → 进入诉讼分诊路径
└── 非诉 → 进入非诉分诊路径
```

### 非诉分诊（第二层）

```
非诉问题 → 属于哪个法律领域？
├── 合同法务 → 分配给 contract-specialist
├── 劳动法务 → 分配给 employment-specialist
├── 知识产权 → 分配给 ip-specialist
├── 数据隐私 → 分配给 privacy-specialist
├── 产品合规 → 分配给 product-legal
├── 行业监管 → 分配给 regulatory-specialist
└── AI合规 → 分配给 ai-governance-specialist
```

### 涉诉分诊（第二层）

```
涉诉问题 → 涉及哪个领域？
├── 民事（合同/债务/物权）→ 分配给 contract-specialist + litigation-strategist
├── 劳动争议 → 分配给 employment-specialist
├── 知识产权侵权诉讼 → 分配给 ip-specialist + litigation-strategist
└── 行政处罚 → 分配给 regulatory-specialist + litigation-strategist
```

## 3. 紧急程度判断标准

### 触发P0的条件（必须立即响应）

1. **监管介入**：证监会/银保监会/市场监管部门正式调查通知
2. **司法查封**：法院冻结账户/查封资产
3. **员工群体性事件**：50人以上劳动仲裁/投诉
4. **重大合同违约**：合作方发出解除通知，可能引发连锁反应
5. **重大数据泄露**：个人信息泄露涉及监管报告义务
6. **刑事风险**：涉嫌商业犯罪（职务侵占/诈骗等）

### 触发P1的条件

1. 对方已提起诉讼（收到传票）
2. 知识产权侵权通知（需要评估是否下架/应诉）
3. 单一员工劳动仲裁（可能引发群体效应）
4. 合同履行出现重大争议
