# 产品法务知识库

> 企业法务产品法务（product-legal）的本地知识手册。用于产品法律风险识别和合规审查。

---

## 1. 产品法律风险识别方法

### 风险识别框架

| 风险类型 | 识别维度 | 常见风险点 |
|----------|----------|-----------|
| **产品安全** | 设计缺陷/生产缺陷/警示缺陷 | 结构安全隐患、材料问题、使用说明不足 |
| **广告合规** | 虚假宣传/夸大宣传/诋毁竞品 | 绝对化用语、功效夸大、对比广告 |
| **消费者保护** | 知情权/选择权/退换货 | 信息披露不充分、格式条款不公 |
| **价格合规** | 价格欺诈/垄断定价 | 虚构原价、价格操纵 |

### 产品责任风险矩阵

| 风险等级 | 发生概率 | 潜在损失 | 建议 |
|----------|----------|----------|------|
| 高风险 | >20% | >500万 | 立即整改，投保产品责任险 |
| 中风险 | 5~20% | 100~500万 | 3个月内整改，建立追溯机制 |
| 低风险 | <5% | <100万 | 例行监控，持续优化 |

## 2. 广告法合规审查

### 禁止性用语（《广告法》第9条）

- 国家标志、国旗、国徽、国歌
- "国家级"、"最高级"、"最佳"等绝对化用语
- 淫秽、色情、赌博、迷信、恐怖内容
- 民族、种族、宗教、性别歧视
- 妨碍环境、自然资源保护

### 功效宣称合规边界

| 宣称类型 | 合规要求 | 违规示例 |
|----------|----------|----------|
| **医疗功效** | 需有临床试验证明/药监局批件 | 普通食品宣称"治疗疾病" |
| **绝对化宣称** | 禁止"最好"、"第一"等 | "全世界最好的面膜" |
| **比较广告** | 需有事实依据，不得诋毁对手 | "比XX品牌强10倍"（无依据） |
| **数据宣称** | 需注明数据来源/测试条件 | "XX%用户反馈有效"（无来源） |

### 代言合规要求

| 代言人类别 | 限制 |
|------------|------|
| **不满10周岁未成年人** | 禁止代言 |
| **不满16周岁未成年人** | 禁止代言医疗、药品、医疗器械、保健食品 |
| **未使用过商品/服务的自然人** | 禁止代言 |
| **专业人员（医生、专家等）** | 应真实使用过，涉及专业领域需持相关资质 |

## 3. 消费者权益保护

### 消费者知情权保障

**必须披露的信息**：
1. 价格/费用（含运费、保修费等）
2. 数量/规格/型号
3. 生产日期/保质期（食品、药品）
4. 安全注意事项
5. 售后服务（退货、保修）

### 无理由退货权（《消保法》第25条）

**不适用无理由退货的商品**：
1. 消费者定作的
2. 鲜活易腐的
3. 在线下载或拆封的数字化商品（音像制品、软件等）
4. 交付的报纸、期刊
5. 其他根据商品性质不宜退货的商品（需事先告知）

### 格式条款合规（《消保法》第26条）

- 不得利用格式条款限制消费者合法权利
- 不得利用格式条款加重消费者责任
- 不得利用格式条款排除消费者权利
- 对格式条款的理解有争议时，按有利于消费者的解释

## 4. 反垄断法相关价格合规

### 垄断协议禁止

**横向垄断协议**（《反垄断法》第17条）禁止：
- 固定或变更商品价格
- 限制商品生产或销售数量
- 分割销售市场或原材料采购市场
- 限制新技术/新产品开发
- 抵制交易

**纵向垄断协议**：固定转售价格、限定最低转售价格（需个案评估）

### 价格欺诈行为

| 欺诈类型 | 示例 |
|----------|------|
| 虚构原价 | 标注"原价"但从未按此价格销售过 |
| 虚假优惠 | 先提价后打折，实际未优惠 |
| 模糊标价 | 仅标注总价，不注明单价/单位 |
| 重量欺诈 | 标注含包装重量而非净含量 |
