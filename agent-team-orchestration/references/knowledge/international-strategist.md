## 1. 各国际平台运营规范

### YouTube运营

| 参数 | 标准值 | 说明 |
|------|--------|------|
| **视频时长** | 8~15分钟最佳 | YouTube偏好长视频（观看时长是核心指标） |
| **缩略图尺寸** | 1280×720px | 16:9，文字不超过6个词 |
| **标题长度** | 50~60字符 | SEO友好，前30字符最重要 |
| **标签数量** | 3~8个 | 包含核心词+长尾词 |
| **片尾字幕** | CTA+相关视频推荐 | 增加订阅和观看时长 |

### Instagram运营

| 参数 | 标准值 | 说明 |
|------|--------|------|
| **图片尺寸** | 1080×1080（方形）或 1080×1350（竖版） | Reels建议1080×1920 |
| **Reels时长** | 15~60秒最佳 | 可上传90分钟长视频 |
| **Hashtag** | 5~15个 | 1个主标签+4个相关+10个细分 |
| **最佳发布时间** | 美国东部时间11:00~13:00 或 19:00~21:00 | 参考目标受众时区 |

### TikTok运营

| 参数 | 标准值 | 说明 |
|------|--------|------|
| **视频时长** | 15~60秒最佳 | 算法偏好短平快 |
| **最佳完播率** | >50% | 超过30秒的视频完播率更重要 |
| **互动率目标** | >8% | 短视频平台用户更活跃 |
| **发布频率** | 每天1~3条 | 多账号矩阵效果更好 |

## 2. 跨境内容本地化方法

### 本地化四维度

| 维度 | 操作要点 | 注意事项 |
|------|----------|----------|
| **语言** | 目标市场本地语言 | 不是简单翻译，语法/表达要地道 |
| **文化** | 避免文化禁忌 | 注意手势/颜色/数字在不同文化中的含义 |
| **热点** | 使用当地热门话题 | 不同时区的节假日/事件 |
| **平台风格** | 各平台内容调性差异 | YouTube vs TikTok风格差异大 |

### 文化差异风险清单

| 文化要素 | 中国习惯 | 西方/其他文化 | 风险程度 |
|----------|----------|--------------|----------|
| **数字** | 6/8/9为吉 | 13为凶 | 高（需避免） |
| **颜色** | 红色喜庆 | 白/黑用于丧事 | 高 |
| **动物** | 龙为吉祥 | Dragon负面印象多 | 中 |
| **食物** | 特定食材 | 宗教/素食文化 | 中 |
| **政治/宗教** | 避免 | 绝对避免 | 极高 |

## 3. 多语言SEO

### YouTube多语言SEO

**标题本地化**：
- 不是翻译，而是重新SEO优化
- 参考目标语言搜索习惯
- 包含目标语言热门关键词

**描述本地化**：
- 前150字符最重要（Google摘要限制）
- 加入目标国家热门搜索词
- 添加时间戳（可被搜索）

**字幕策略**：
- 提供CC字幕（YouTube自动翻译+人工校正）
- 关键内容可上传独立字幕文件

### 多语言关键词工具

| 工具 | 用途 |
|------|------|
| Google Trends | 各语言搜索趋势对比 |
| AnswerThePublic | 本地语言问答型关键词 |
| YouTube Search Suggest | YouTube搜索下拉词 |
| Ubersuggest | 多语言SEO关键词分析 |

## 4. 跨平台运营矩阵

### 国际平台矩阵建议

| 内容类型 | 主平台 | 分发平台 | 本地化程度 |
|----------|--------|----------|------------|
| 短视频/娱乐 | TikTok | YouTube Shorts/Instagram Reels | 中度本地化（音乐/字幕） |
| 知识/教程 | YouTube | TikTok/B站（海外华人） | 高度本地化 |
| 生活方式/Vlog | YouTube/Instagram | TikTok | 高度本地化 |
| 品牌内容 | Instagram | YouTube | 中度本地化 |

### 各平台内容差异化

| 平台 | 内容风格 | 制作重点 |
|------|----------|----------|
| YouTube | 专业、深度、系列化 | 缩略图+标题SEO |
| Instagram | 视觉优先、生活方式 | 高质量图片+Reels |
| TikTok | 轻松、真实、跟风 | BGM选择+热点跟风 |
| Facebook | 社区感、长文 | 图文帖子+群组 |
