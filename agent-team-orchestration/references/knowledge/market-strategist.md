## 一、PE/PB 估值百分位计算

### 1.1 PE估值百分位

**公式**：
$$PE百分位 = \frac{\text{当前PE在历史序列中的排名}}{\text{历史数据总个数}} \times 100\%$$

**计算步骤**：
1. 收集该指数/个股历史PE序列（通常取过去10年）
2. 去除极端值（PE < 0 或 PE > 200）
3. 升序排列
4. 计算当前PE的位置百分比

**Python实现参考**：
```python
pe_percentile = (pd.Series(historical_pe) < current_pe).mean() * 100
```

### 1.2 PB估值百分位

**公式**：
$$PB百分位 = \frac{\text{当前PB在历史序列中的排名}}{\text{历史数据总个数}} \times 100\%$$

### 1.3 估值百分位判断标准

| 百分位区间 | 估值状态 | 操作建议 |
|-----------|---------|---------|
| < 20% | 极度低估 | 分批建仓 |
| 20%-40% | 偏低估 | 逐步加仓 |
| 40%-60% | 适中估值 | 持有观望 |
| 60%-80% | 偏高估 | 逐步减仓 |
| > 80% | 极度高估 | 止盈/清仓 |

## 二、股债收益率比模型（股票风险溢价）

### 2.1 核心公式

$$股票风险溢价 = \frac{1}{PE} - 债券收益率$$

或：

$$股债收益率比 = \frac{股票收益率（1/PE）}{债券收益率}$$

### 2.2 判断标准

| 股债收益率比 | 市场含义 | 信号 |
|-------------|---------|------|
| > 2.0 | 股票极具吸引力 | 买入信号 |
| 1.5 - 2.0 | 股票相对债券吸引力较强 | 偏多 |
| 1.0 - 1.5 | 股票吸引力中性 | 中性 |
| < 1.0 | 债券更有吸引力 | 减仓信号 |

### 2.3 实际应用公式

$$ERP = \frac{1}{沪深300PE} - 10年期国债收益率$$

- **ERP > 2%** → 股票资产明显低估
- **ERP < 0** → 股票资产明显高估

## 三、市场情绪指标

### 3.1 融资融券余额

**融资余额**：
$$融资余额 = \sum_{i=1}^{n} 投资者_i 融资买入金额$$

**融券余额**：
$$融券余额 = \sum_{i=1}^{n} 投资者_i 融券卖出数量 \times 当前股价$$

**判断标准**：
| 融资余额/流通市值 | 市场情绪 |
|------------------|---------|
| < 3% | 情绪低迷 |
| 3%-5% | 情绪中性 |
| 5%-8% | 情绪偏高 |
| > 8% | 情绪过热/杠杆牛 |

**融资净买入变化**：
$$融资净买入 = 今日融资余额 - 昨日融资余额$$

### 3.2 北向资金

**净买入计算**：
$$北向资金净流入 = \sum_{i=1}^{n} (买入额_i - 卖出额_i)$$

**判断标准**：
| 单日净流入 | 信号强度 |
|-----------|---------|
| > 100亿 | 强势买入 |
| 50-100亿 | 积极买入 |
| 10-50亿 | 温和买入 |
| -10-10亿 | 观望 |
| < -50亿 | 积极卖出 |
| < -100亿 | 恐慌卖出 |

### 3.3 换手率

$$换手率 = \frac{\text{当日成交股数}}{\text{流通股本}} \times 100\%$$

**判断标准**：
| 全市场换手率 | 市场状态 |
|-------------|---------|
| < 1% | 冰点 |
| 1%-2% | 低迷 |
| 2%-3% | 活跃 |
| 3%-5% | 高度活跃 |
| > 5% | 投机过热 |

## 四、大小盘轮动规律

### 4.1 国证1000/国证2000定义

- **国证1000指数**：A股市场市值最大、流动性最好的1000只股票
- **国证2000指数**：剔除国证1000后，市值最大的2000只股票

### 4.2 相对强弱指数

$$大小盘比值 = \frac{国证2000收盘价}{国证1000收盘价}$$

$$RS\_Score = \frac{大小盘比值 - 20日均线}{20日标准差}$$

### 4.3 轮动规律判断

| 宏观环境 | 相对强势 | 逻辑 |
|---------|---------|------|
| 经济复苏初期 | 大盘价值 | 基建/银行/周期 |
| 经济复苏中后期 | 小盘成长 | 弹性更大 |
| 流动性宽松 | 小盘/成长 | 资金成本低 |
| 流动性收紧 | 大盘价值 | 防风险优先 |
| 风险偏好提升 | 小盘/主题 | 题材炒作 |

## 五、市场PE中位数计算方法

### 5.1 计算公式

$$PE_{中位数} = 中位数\left(\frac{P_1}{EPS_1}, \frac{P_2}{EPS_2}, ..., \frac{P_n}{EPS_n}\right)$$

### 5.2 计算步骤

1. 收集所有股票的PE（剔除PE<0或PE>500的异常值）
2. 升序排列
3. 取中位数：若n为奇数，取第(n+1)/2个；若n为偶数，取第n/2和第n/2+1的均值

### 5.3 判断标准

| 市场PE中位数 | 市场估值 |
|-------------|---------|
| < 20倍 | 极度低估 |
| 20-30倍 | 偏低估 |
| 30-40倍 | 适中 |
| 40-50倍 | 偏高估 |
| > 50倍 | 泡沫区域 |

## 七、快捷工具

```python
# PE百分位计算示例
def pe_percentile(current_pe, historical_pes):
    return (sum(h <= current_pe for h in historical_pes) / len(historical_pes)) * 100

# 股债收益率比
def stock_bond_ratio(pe, bond_yield):
    return (1/pe) / bond_yield

# ERP计算
def equity_risk_premium(pe, bond_yield):
    return (1/pe) - bond_yield
```
