# AI 内容创作团队 SOP

## 基本信息

- **团队名称**：ai-content-creator（AI内容创作）
- **Agent 数量**：5
- **触发词**：AI内容创作/多模态内容/视频生成/内容创作

## 团队架构

| Agent | 角色 | 职责 |
|-------|------|------|
| content-director | 内容总监 | 制定内容计划、协调各环节、把关质量 |
| scriptwriter | 脚本撰写 | 撰写视频/图文脚本，规划叙事结构 |
| visual-artist | 视觉艺术家 | 生成图像、设计视觉元素 |
| video-editor | 视频剪辑师 | 将素材剪辑成片，做适配处理 |
| platform-adapter | 平台适配 | 根据平台特性调整内容格式和风格 |

## SOP 流程

### Phase 1：内容计划

**输入**：用户需求描述
**输出**：`content-plan.md`
**目的**：明确内容方向、目标受众、平台选择

**步骤**：
1. content-director 分析用户需求
2. 确定内容类型（视频/图文/多模态）
3. 选定目标平台（抖音/小红书/YouTube 等）
4. 编写内容计划文档

### Phase 2：素材生成

**输入**：content-plan.md
**输出**：脚本 + 视觉素材
**目的**：生成完整的创作素材

**步骤**：
1. scriptwriter 撰写完整脚本
2. visual-artist 生成配套图像
3. 素材汇总到 artifacts/{team_id}/

### Phase 3：制作剪辑

**输入**：素材
**输出**：粗剪版本
**目的**：完成内容主体制作

**步骤**：
1. video-editor 将素材剪辑成片
2. 加入转场、音效、字幕
3. 输出粗剪版本供审核

### Phase 4：平台适配

**输入**：粗剪版本
**输出**：各平台最终版本
**目的**：适配不同平台的格式要求

**步骤**：
1. platform-adapter 分析各平台特性
2. 调整时长、比例、封面
3. 生成各平台最终版本

## 阶段关卡（Phase Gates）

| 关卡 | 通过条件 | 失败处理 |
|------|----------|----------|
| PG1-内容计划 | content-plan.md 包含完整的类型/受众/平台信息 | 返回 Phase 1 重新规划 |
| PG2-素材生成 | 脚本已定稿，视觉素材至少 3 个可用版本 | 补充素材或调整方向 |
| PG3-制作剪辑 | 粗剪版本时长误差 <10%，叙事完整 | 重剪或修改脚本 |
| PG4-平台适配 | 各平台版本格式符合要求 | 调整格式参数 |

## 交接协议

### 产出物格式

```yaml
---
team_id: {team_id}
agent_id: content-director
role: 内容总监
phase: planning
status: done
findings: |
  内容类型：短视频
  目标平台：抖音、小红书
  目标受众：18-25岁女性
---
```

### 验收字段

- `phase`：当前完成的阶段
- `status`：done/in_progress
- `findings`：核心产出描述
